Design System Inspiration of PlayStation

A three-surface marketing system organized around alternating black, white, and PlayStation Blue chapters that scroll past the viewer like a console launch trailer. PlayStation SST display copy at signature weight 300 (light) gives the chrome an airy, premium feel; bright PlayStation Blue (#0070d1) carries every primary CTA as a fully-rounded pill; commerce orange (#d53b00) anchors store actions. The system never decorates — imagery does all the heavy lifting.

Touch Targets

  • WCAG AAA: button-primary and button-commerce at 48px height with 28px padding (~48×100px tappable).
  • Text input: 48px height with 16px padding.
  • Support search bar: 56px height with 24px padding.
  • Carousel paddle: exactly 48×48 circular.
  • Filter pill: ~36–40px extends to 44px tappable via padding.

Collapsing Strategy

  • Primary nav: desktop horizontal cluster → tablet hamburger drawer at 768px.
  • Sub-nav: desktop horizontal anchor row → tablet horizontal scroll → mobile select dropdown.
  • Game carousel: 4-up → 3-up → 2-up at 1024 and 768px.
  • Hero bands: stay full-bleed; internal column reflows from 2-col to single-col.
  • Section padding: 96px → 64px → 48px.
  • Hero headline: 54px → 44px → 32px → 28px down the breakpoint stack.
