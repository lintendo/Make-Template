# 18-sketch101 页面规格说明

## 1. 业务与功能

### 1.1 页面/元素定位与核心目标
- **类型**：课程聚合与详情页 (Course Detail & Landing Page)
- **目标**：展示以 "Sketch 101" 为代表的官方新手视频课程体系，激发用户参与学习的兴趣。提供课程的完备信息（大纲、导师、预期时长），并在整个阅读过程中维持一个吸顶的“开始课程”行动号召。

### 1.2 功能清单
- **英雄首屏视频位 (Hero Video Section)**：
  - “SKETCH COURSES” 的次级分类提示和巨型主副标题。
  - 核心视觉元素为一张拥有极强圆角 (rounded-2xl) 的超宽巨幅视频封面图，居中覆盖了一个带有半透明黑底的白三角播放按钮。
  - 背景采用了充满生机感与呼吸感的大型渐变光斑 (Orbs 背景)。
- **左右不对称阅读流 (Two-Column Asymmetric Layout)**：
  - 页面主体为 `网格 8:4` 左右分栏。
  - **左侧为主图文阅读区**：
    - 大纲描述、技能树 (What you'll learn)、课程目录列表 (Course lessons) 以及硬件/前置条件。
    - “导师 (Mentor)” 的详细图文介绍。
  - **右侧为悬浮信息卡片 (Sticky Info Card)**：
    - 一张具有从柔和暖橙色渐变到纯白背景的高保真信息卡片。整合了课时、总时长、语言和费用状态，底部为一个极强的黑底促动按钮（Start course）。
    - 具有粘性定位 (Sticky)，跟随用户向下滚动阅读左侧长文。
- **章节列表手风琴/数据表 (Chapter & Lessons List)**：
  - 采用灰白分明的条带设计。表头 (Header) 为浅灰色带有章节概览，内部的课程内容使用白底，右侧附带准确的时长标记以及橙色的播放小图标。
- **底部的订阅预告 (Newsletter CTA)**：
  - “Sketch 102 is coming”，背景散落着浅灰色的线框简笔画图案（如调色板、书本、准星等）。

## 2. 视觉特征与细节提取

### 2.1 右侧的 Sticky Meta Card (信息大卡片)
这是本页设计的精髓之一。它并没有使用冷冰冰的白底黑字，而是使用了 `bg-gradient-to-b from-[#FFF5E5] to-white` 这种带有极强情绪引导的暖色调渐变，让人产生“欢迎、简单、友好”的心理暗示。每个信息条目（Lesson / Time / Language）都使用了 `border-b border-gray-100` 的极细分割线，并搭配精致的小粗体 Icon（如播放符、时钟、标签等），构建出了高层级的阅读体验。

### 2.2 章节课程列 (Course lessons list)
章节的行条目呈现出严谨的网格系统。虽然没有明确的竖线分割，但课时编号（如 1.1）、课时名称、时长、播放图标严格垂直对齐。使用了非常浅的背景色（`#F9F9F9`）来包裹整个 Chapter 区域。

## 3. 实现要求
- 导航栏和通用底部保持原样一致。
- 右侧的课程卡片必须具有 `sticky top-24` 属性，确保在页面滚动时不会消失。
- 大量的背景装饰物，包括顶部的彩色模糊光斑背景以及底部的浅灰色线稿插图，都将通过 CSS / SVG Absolute 定位的形式进行注入。
- “What you'll learn” 的打钩符不使用普通的文本符号，而是注入小号的加粗带有橙色（或黑色）的主题色 Tick mark 来提升精致感。
