# 22-contact 页面规格说明

## 1. 业务与功能

### 1.1 页面/元素定位与核心目标
- **类型**：客服与支持工单页 (Customer Support / Contact Form)
- **目标**：为用户提供反馈问题、提交 Bug、寻求商务帮助的统一入口。通过长表单收集必要的发件人与上下文信息。

### 1.2 功能清单
- **全局导航 (Global Nav)**：标准黑色 / 浅色模式自适应导航（取决于背景层叠）。在此页面中是经典的底色为白色，按钮黑色。
- **左侧内容区 (Form Area)**：
  - 极具视觉冲击力的大标题 `Contact Us`。
  - 两段有情感的引导性文字（说明响应速度、支持查阅实时 `status` 状态中心）。
  - **复杂表单系统**：
    - Email (Email Input)：带附加说明 `(So we can reply to you)`
    - Topic (Select Dropdown)：选择问题所属的业务类型
    - Subject (Text Input)：工单副标题
    - Full description (Textarea)：多行文本输入，有很大高度。
  - **附件上传提示 (Attachment Hints)**：纯文本的说明，提示用户使用诸如 Dropbox、Google Drive 等第三方网盘上传大文件，文本内含带有黑色硬下划线的链接。
  - **隐私协议复选框 (Checkboxes)**：两组 Checkbox（同意服务条款、订阅资讯）。
  - **核心操作 (Submit Action)**：沉稳厚重的纯黑色“Send”按钮。
- **右侧视觉焦点 (Right Decorator Graphic)**：
  - 充满屏幕右侧的超大型环境装饰物：一个巨大的 3D 感紫红色气泡（Speech Bubble）。它由一个巨型的球体和下方不规则的“尾巴”组成。
  - 在巨大气泡的下方，有一条长长的柔光感紫色柱状背景块顺延到底部。

## 2. 视觉特征与细节提取

### 2.1 极度厚重的黑色按钮与下划线文本
该页面的超链接设计与通常的蓝色下划线不同，它使用了非常 Old-School 且极客的粗黑下划线设计（`border-b-2 border-black` 或 `text-decoration-thickness`）。
提交按钮也彻底抛弃了渐变光泽，退回到最基础暴力的纯黑色实心按钮，传达出一种硬核的“执行/发送”力量感。

### 2.2 软球状紫红 3D 气泡（Magenta 3D Bubble）
右侧抢眼的视觉对象是一个像紫红色泡泡糖一样的气泡。要通过纯代码来实现它，需要非常老练地叠加多个 `radial-gradient`（包含偏白色的高光色和极深的紫黑阴影色）到圆形的 `div` 容器上，以制造出令人惊讶的体积感、漫反射和环境遮蔽光。最下方带两个小尾巴的设计也可以用纯 CSS 的三角形 / 贝塞尔曲线修饰。

## 3. 实现要求
- 页面必须按照 `lg:w-1/2` 的比例切分为左（表单）右（插图）结构，在移动端需自动垂直堆叠并隐藏复杂插画或缩小插画。
- 所有的表单组件要具有典型的 Sketch 默认 UI 质感：浅灰色 `border-gray-200` 描边，获取焦点时轻微的黑边或外发光。
- 通用 Footer 复用。
