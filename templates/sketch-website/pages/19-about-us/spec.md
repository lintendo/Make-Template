# 19-about 页面规格说明

## 1. 业务与功能

### 1.1 页面/元素定位与核心目标
- **类型**：关于我们 / 品牌故事页 (About Us / Brand Story Page)
- **目标**：向公众、媒体和用户讲述 Sketch 的创业历程、核心价值观（Core Beliefs）以及开放透明的品牌形象，同时供给公关媒体下载官方视觉资产（Press Kit）。

### 1.2 功能清单
- **全屏浸入式英雄首屏 (Immersive Hero Area)**：
  - 左对齐的长版品牌宣言，讲述产品起源。
  - 极宽广的横向时间轴 (Timeline)，串联了“2010年发布”、“Apple Design Award”、“1百万用户”、“A轮融资”等重要里程碑。
  - 背景延用了招聘页的 Sketch 标志性重工视觉：巨型橙黄色系多边形几何切割。
- **品牌信条网格 (Beliefs Grid)**：
  - 两栏排版，无多余装饰，纯文本展示五条核心信条。
- **客户背书跑马灯 (Trusted By Bar)**：
  - 典型的 “Trusted by” 客户 Logo 墙，包括 Google, Stripe, Xbox 等世界级科技公司。
- **联合创始人介绍区块 (Co-Founders Bio)**：
  - 一组使用复杂 CSS 堆叠（同心圆环、斜切透视、悬浮卡片）的双人合影组件。
  - 右侧配合了展示远程工作规模的核心数据看板（220 / 43 / 0）。此块布局与招聘页保持高度的一致性体验。
- **媒体资源库 (Press Information)**：
  - 三列布局的高保真卡片。每张卡片上半部分为炫致的渐变色背景与资产缩略图，下半部提供一键下载按钮，供应了标准 Logo、应用图标和界面截屏压缩包。

## 2. 视觉特征与细节提取

### 2.1 极度精简的单线时间轴 (Minimalist Timeline)
位于首屏下方的时间轴设计得非常巧妙：它并没有使用笨重的卡片框定，而是完全依托悬浮在空间中的一根细发丝一样的水平参考线 `1px solid border-gray-200`，所有里程碑事件（点和信息标签）通过绝对定位吸附在这根线上。各个节点有的在上方，有的在下方交错排布，维持呼吸感。

### 2.2 共用的立体合照图床 (Shared 3D Profile UI)
这里的创始人头像排版复用了“Who We Are”的景深范式：
- 背景必须是细如蛛丝的嵌套圆框。
- 特色的 Sketch Orange 色调的绝对实体圆点。
- 主图的微弱倾角（例如 `transform -rotate-2`），与水平摆正的高斯模糊简介白卡产生结构张力。

### 2.3 “大数字”统计法 (Big Number Typography)
对于 `220`、`43`、`0` 这三个数据，使用了极窄页宽里的极简排布，字号甚至突破常规的 `text-[64px]` 黑体量级。

## 3. 实现要求
- 时间轴使用相对定位容器 `relative` 取代写死的表格，保证响应式布局下也能用 Flex 控制其间距与上下交错。
- Logo 墙中的 SVG 图标用黑色纯色或图片占位符示意。
- Press Kit 三张卡片的上半身必须用代码重组纯粹的 CSS 渐变高光和阴影，还原“艺术品”摆放台的质感。
