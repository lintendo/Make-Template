# 25-newsletter 页面规格说明

## 1. 业务与功能

### 1.1 页面/元素定位与核心目标
- **类型**：Newsletter (邮件通讯订阅) 落地页。
- **目标**：说服用户订阅 Sketch 的官方电子报 "Beyond the Canvas"。不仅提供邮件订阅入口，还展示了过往所有精彩期刊的归档列表，增强订阅动机。

### 1.2 功能清单
- **全局通知横幅 (Promo Banner)**：最顶部的极简文字横幅。
- **英雄头图区 (Hero Section)**：
  - **背景**：采用极其明亮灿烂的暖黄色渐变光波，并且底部被切出了一个犀利的**上扬对角线 (Diagonal Cut)**。
  - **左侧**：一本使用代码重构的、极具 3D 物理质感的 "Beyond the Canvas" 实体书（代表电子报的实体形态），书页微微掀开，带有一张绿色的 `ISSUE 01` 标签和蚀刻画板图样。
  - **右侧**：订阅表单。包括大段宣传语、Email 输入框、带定制颜色的 Subscribe 按钮（深灰褐色/泥黄色）、同意条款复选框。
- **双栏价值主张 (Value Props)**：两个图文并茂的短段落，说明订阅的好处（人工精选资源、最新动态）。
- **往期回顾 (Previous Issues Archive)**：
  - 左侧固定的排版区域：巨大的标题 `Catch up on previous newsletters` 和引语。
  - 右侧滚动的列表区：陈列了 Issue 08 到 Issue 01 共 8 期过往电子报。每一期都配有一个专属配色的精美圆角插画和生动的标题。
- **末端绿色转化区块 (Bottom CTA Banner)**：
  - `Create your Workspace for free`。基于柔和圆角盒子的深绿色弥散光束背景，呼吁用户立刻注册账号。
- **标准主站页脚 (Standard Footer)**。

## 2. 视觉特征与细节提取

### 2.1 物理厚度书本的 CSS 建模
原图中有一本非常逼真的特刊书籍作为头图。我要在不依赖原图的情况下，使用纯 CSS 来构造这本 3D 杂志。将使用 `transform` 体系的 `rotateY`、`rotateZ` 和 `skew`，配合深重多层的 `box-shadow` 制造出书脊厚度和右侧纸张的堆叠感。封面的 "Beyond the Canvas" 进行绝对定位打字排版，并画一个红色的小画板。

### 2.2 底部对角线切割 (Clip-Path Diagonal Cut)
黄色天空首屏的底部并非直线，而是强烈的斜切。技术上可以直接使用 CSS `clip-path: polygon(0 0, 100% 0, 100% 85%, 0 100%)` 完美裁出这个利落边缘。

### 2.3 精致的过往期刊图标列阵
右侧在展示 Issue 01~08 时，每一个 List Item 左侧都有一个 `60x60` 尺寸的高亮色块，里面散落着不同的小星星、苹果、画板等矢量碎皮。我将用 CSS 的基本图形和网格拼接，给每一个 Issue 随机分配一种渐变主题色（绿、蓝、粉、黄等），以营造目不暇接的丰富感。

## 3. 实现要求
- 页面需要在 Desktop 端保持这种非常大气的左右切分结构（如往期回顾区的左侧悬停/居中定宽，右面流式列表）。
- 底部 CTA 必须含有非常平滑的绿黄色相间的超大 Radial Gradient。
