# 16-education 页面规格说明

## 1. 业务与功能

### 1.1 页面/元素定位与核心目标
- **类型**：教育计划/公益豁免申请页 (Education / Non-profit Landing Page)
- **目标**：向符合条件的学生和教育工作者展示免费获取 Sketch 的流程并引导申请。整体氛围偏向年轻、创意、亲和力（校园氛围）。

### 1.2 功能清单
- **英雄宣言区 (Hero Section)**：
  - “Sketch For Education” 双字体混排标题，强调其非商业的专属属性。
  - 带有一条独特的手绘红色弯曲箭头，指向右侧的主插画区（或下方流程区）。
  - 右侧配有学生元素的 3D 浮雕图（MacBook、笔记本、漂浮文具）叠加霓虹冷暖渐变。
- **黑板操作流程图 (Chalkboard Step-by-Step)**：
  - 采用独具特色的“实木边框黑板”UI 设计：深邃的类黑板底色（暗灰黑带极细微杂色感觉），被一圈原木色边框包裹。
  - 黑板内部是粉笔字质感的箭头，连接着第一步（注册）与第二步（申请）。
- **附加提示卡片 (Notes Grid)**：黑板下方附有 3 个浅灰色的说明卡片，提供设备需求、购买入口等额外信息。
- **课程及支持引流 (Resources)**：包含两张立体白底卡片（带有乐高积木等 3D 元素），推介基础课程与帮助中心。
- **FAQ 问答手风琴 (Common Questions)**：左右两栏布局，左侧大标题，右侧是极简线条风格的问答面板。
- **底部粉蓝天际线 CTA (Bottom Banner)**：一张大画幅的蓝紫粉色“日出/日落天空渐变”横幅卡片，激发用户立即申请的行动。

## 2. 视觉特征与细节提取

### 2.1 书写体混排 (Cursive Script Font)
在此页面中，高频使用了红色/橙红色的手写体（如 *For Education*, *to get started*, *Learn how to apply* 等）。这种非正式的字体样式极大地柔化了软件商业网站的僵硬感，必须在 CSS 中使用 `font-family: cursive, "Comic Sans MS", var(--font-handwriting, sans-serif)` 进行还原，配以适当的倾斜与间距调整。

### 2.2 仿真材质：木框黑板 (Simulated Texture)
页面中部核心流程区是一张黑板：
- `bg-[#2A2B2C]` 的暗系主体加上极为轻微的 `box-shadow: inset` 模拟黑板。
- 外圈使用 `border-[12px] border-[#A67B5B]` (橡木色) 并配以外围大阴影以突出立体感。
- 黑板内部的两步之间的连线（白色手绘波浪箭头）将用内嵌的绝对定位 `<svg>` 矢量图形精确重绘。

## 3. 实现要求
- 手写排版需要与主视觉黑体字形成鲜明的大小对比与颜色打断。
- 英雄区的装饰性手写红字和剪头，要使用 Absolute 定位悬挂在正常文本流的下方。
- 引入可复用的 `<FAQItem>` 以支撑繁重的 Q&A 文档列展示。
- 将图片占位符设计成干净的毛玻璃或者纯 CSS 多彩渐变色块配以简明 Emoji 以保持项目高定的一致感。
