# 17-careers 页面规格说明

## 1. 业务与功能

### 1.1 页面/元素定位与核心目标
- **类型**：企业招聘/人才招募落地页 (Careers Landing Page)
- **目标**：向求职者展示 Sketch 的远程工作文化、核心价值观、员工福利、招聘流程，并提供职位申请入口。传递出开放、自由、无国界的企业精神。

### 1.2 功能清单
- **英雄首屏 (Hero Section)**：
  - 中置式的排版结构，宣示 `Careers at Sketch` 和核心使命。
  - 主动作按钮 `See current openings` 引导至锚点或职位列表。
  - 背景使用温暖的巨型交点几何块（橙/黄/红色调），传递活力。
- **居中浮动锚点导航 (Sticky Page Nav)**：
  - `Who we are`, `Benefits`, `Job openings`, `Our Process` 四个板块的页内快捷跳转菜单，带有 Emoji 级别的高清微型图标。
- **关于我们/核心数据 (Who We Are & Core Values)**：
  - 左侧：员工感言卡片组。通过重叠的头像照片、带阴影的白底寄语卡片和背景的同心圆线框（Concentric wireframes），组成一个典型的见证人组件 (Testimonial UI)。
  - 右侧：亮眼的关键业务数据 `220 PEOPLE`、`43 COUNTRIES`、`0 OFFICES`（强调全员远程），以及具体的价值观长文文本（Freedom and trust, Embracing change...）。
- **福利一览 (Benefits Grid)**：
  - 4列网格排版的福利清单（如弹性时间、设备预算、无限期假等）。使用了轻量的 3D / 写实微图标（如日历、地球仪、海滩伞等）。
- **当前职位空缺 (Job Openings)**：
  - 在淡灰色背景上呈现全职或兼职岗位的列表卡片，分类展示不同团队的空缺情况。
- **面试流程 (Our Process)**：
  - 用两列并排的阅读模式展示了面试的四大关键步骤。

## 2. 视觉特征与细节提取

### 2.1 错落有致的多层员工感言卡片 (Overlapping Testimonial Card)
“Who we are” 旁边的感言卡组件，体现了 Sketch 极强的 UI 设计功力：
- 最底层是同心圆蛛网状的极细 `1px` 灰色圆形线条，打破矩形框的沉闷。
- 中层是一张巨大的正方形圆角人像图，并在右上角叠放一个纯色的橘红大圆圈作视觉平衡。
- 顶层是一张毛玻璃白底的高级高保真卡片，包含了该员工的一段斜体字引用语、名字和职务，外加一个极小的英国国旗作为点缀。整个顶层卡片需要利用 `absolute` 定位脱离文档流产生强烈的景深感（甚至超出了照片边界）。

### 2.2 0 办公室的排版 (Zero Offices Numeric Stats)
利用了超大字号和紧凑的粗体无衬线字体，凸显 `220`、`43`、`0` 的数字。特别是数字 “0” 表达出 100% Remote 的办公哲学，字体加粗和深灰色调构成了有说服力的背书。

## 3. 实现要求
- 首屏复杂的几何背景，可以通过纯 CSS `clip-path: polygon()` 画出巨大的多边形并赋予其亮橙与明黄的渐变，拼接而成。
- 福利部分的图标，可以借助 Emoji 和精心制作的外壳，或者构建一些彩色的块状图代替。
- 工作岗位的卡片应该有一层细微的边框与 `hover:shadow-lg` 交互反馈。
- 面试流程列表通过 `columns-2` 或者 `grid-cols-2` 与小圆点 bullet 实现流式布局。
