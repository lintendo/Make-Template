# 06-blog 页面规格说明

## 1. 业务与功能

### 1.1 页面/元素定位与核心目标
- **类型**：博客内容列表页 (Blog Index)
- **目标**：展示 Sketch 官方的各类动态、教程与社区分享。该页面是内容流的核心枢纽，采用大图单列布局，保证极高的视觉冲击力和易读性。

### 1.2 功能清单
- **定制化导航栏**：整体结构同官网标准导航，但右侧 `Get started for free` CTA 按钮变为了醒目的 **橙色**。
- **博客抬头 (Blog Header)**：左侧超大号标题“Blog”与分类筛选器“All Posts ⌄”；右侧为社交与工具区（包含邮件、RSS订阅、Twitter 链接及搜索框图标）。
- **文章大卡片流 (Single-Column Article List)**：
  - 每篇文章占据一行，宽度与内容容器一致。
  - 结构包含：高清 16:7 宽屏封面图 -> 分类标签（青绿色文本）与日期（灰色带小图标）构成的元信息行 -> 大字号加黑文章标题 -> 灰色正文摘要。
- **内联订阅横幅 (Inline Newsletter Banner)**：在滚动阅读流中穿插的一个无缝灰色底框，包含邮件订阅引导，带有一枚橘色提交按钮。
- **特殊无图文章卡片 (Release Note Text-Card)**：用于更新通知（例如新 Mac App 发布），没有大图封面，左侧是一个 Sketch 钻石小图标志，右侧是更新内容正文与两个功能按钮（“Release notes”与“Documentation”）。
- **加载更多 (Load More)**：列表底部的全宽白色边框按钮。
- **全局 Footer**：标准的版权和导航链接。

## 2. 视觉特征与细节提取

### 2.1 配色方案
- **强调色转换**：本页大量将全局常用的蓝色/黑色 CTA 转换为 **橙色 (#EA580C / Sketch Orange)**，体现出博客区不同的活力视觉规范。
- **分类标签字色**：卡片上方的 `SKETCH TUTORIALS`、`DESIGN TUTORIALS` 等标签文字采用了辨识度较高的小写大写字母 (Uppercase) 和 **青绿色 (#10B981 或 #0D9488)**。
- **主文本**：标题坚持纯黑极粗体（`#1D1C1B`），摘要为经典的灰色（`#585754`）。

### 2.2 布局规律
- 整个列表被收束在一个居中的容器内（约 `max-w-4xl` 或 `max-w-5xl`），确保阅读正文和看大图时的视线焦点集中。
- 图片与文字信息之间有严格的节奏感：`图 -> 间隔 -> Meta行 -> 间隔 -> 标题 -> 间隔 -> 摘要 -> 大底部间隔 (mb-24)`。

## 3. 实现要求
- 编写一套可复用的 `<ArticleCard />` 组件来渲染绝大部分纯图文博客。
- 编写一个特异化的 `<ReleaseNoteCard />` 来应对中间出现的纯文字通告。
- 插入横条形态的 `<InlineSubscribe />` 横幅模块。
