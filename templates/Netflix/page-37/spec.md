# Netflix Core Component: Deep-Scroll Episodes Array (Page 37)

## Logical Block Architecture

### 1. `EpisodeRow` Dynamic Component Grid
Built a highly dense UI row generator mapping the core metadata stream: Cover Art, Episode Enumeration, Duration metric, Action Payload trigger (Download/Watched flag), and linguistic synopsis drop. The internal grid forces strict limits keeping rows responsive.

### 2. Multi-Scene Procedural Engine Array
Four entirely separate `16:9` cinematic scenes procedurally mapped in SVG:
- **Episode 1**: Train interior geometry matching rigid blue bench setups. Used layered `<path>` components for deep window depth-of-field.
- **Episode 2**: Handled a dense 6-character room composition locking background entities in simplistic `<circle>` forms to simulate camera depth blur, focusing focal point nodes purely on Julia Montague's face profile pathing.
- **Episode 3**: Emulated a highly cinematic dual close-up layout. Dropped a huge `filter: blur(25px)` `<circle />` dead-center background to introduce the anamorphic lighting glow separating the two `<path>` profile actors.
- **Episode 4**: Drafted a symmetrical 3-character dark interrogation arrangement mirroring visual geometry weight evenly.

### 3. Conditional Node Binding (`isDownloaded`)
Implemented boolean mapping converting standard SVG download geometry mapping `M12 3v13m-5-5l5 5 5-5` (Netflix arrow format) into absolute Blue Flag status `<div className="bg-[#0A84FF]">` on identical component calls.

### 4. Zero-Friction Navbar Mutation 
Hardcoded the sticky Modal header background into a rigid `bg-[#111111]/95` context, signifying absolute depth transition from the former transparent gradient mode observed in earlier frame ranges.
