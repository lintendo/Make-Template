# Slack Direct Messages Drawer

## 📋 概述

### 1. 核心业务与场景定位

这是 Slack APP 原型中的 "Direct Messages"（私信/频道目录）左侧抽屉菜单页面。展示了当用户从主聊天界面向右滑动时，通过底层透显模式拉出私信面板的视觉形态。左侧约占据 88% 的宽度为紫色的原生导航面板，右侧为被向右推离的主工作台视图局部截图，高度还原了老版 Slack 移动端的空间交互层级结构。

### 2. 用户目标与关键任务

- **导航切换**：快速浏览最近联系人和机器人，点击切换会话。
- **状态管理**：查看联系人在线状态（绿点）。
- **空间感知**：感知左侧菜单与主工作区（被挤到右边的小白条）的层级关系与切换动画停滞态。

## 🧩 内容规划

### 1. 信息架构与模块划分

页面采用典型的侧滑架构，横向分界为两部分：主功能抽屉 (Left Drawer) 与主背景透视 (Right Peek)

**左侧主功能抽屉（暗紫背景色区）**
1. **iOS 状态栏**：全局置顶的电池、信号、运营商与时间栏，该模块部分延伸入紫色区域。
2. **顶部 Header**：包含居中的粗体文字 "Direct Messages" 以及右侧的一键编辑/发消息的线框铅笔图标。
3. **私信列表 (RECENT)**：
   - Section 标题栏 ("RECENT"，灰色大写字样)。
   - **联系人 1 - Slackbot**：经典红绿蓝黄多色块拼接的 Slackbot 头像，并配有右下角的绿色小圆点（Online 状态）。拥有用户名、日期和部分消息回执的二级摘要显示。
   - **联系人 2 - huang (you)**：同第一界面（page-01）一致的本人手绘 Viking 头像，在线状态，并携带 `(you) 🌴` 名称后缀。
4. **底部导航**：
   - 带有三个进度圆点的椭圆形翻页指示器容器，靠右第三个点高亮，表明已滑动至对应空间面板。
   - 纯白色粗线的 iOS Home 指示条。

**右侧层透显（亮灰/白背景区）**
- 位于屏幕右侧约 12% 宽度的透视裂隙（`w-[89%]`左边界），带有巨大的阴影投射进自身，营造出底层紫色抽屉向上推移盖住它的错觉，或是自身右滑盖在紫色层上的纵向交错空间感。
- 显示被截断的 Slack hash logo，断裂的文字，切了一半的绿色联系人、红色切断图标。
- 展现处于主列表底部的多个被截断头像区，以及 `@` 标记图标堆叠。该区域主要作用是视觉关联和动画隐喻。

### 2. 数据与内容来源

本地 Hardcoded Mock Data。头像统一采用内部 `<svg>` 手工绘制占位（Slackbot、Viking）。截断内容亦采用硬编码的字母(`Fe/Ho`, `B/w`) 以 1:1 像素级拟真排版。

## 📐 布局与结构

- 背景承载容器 `flex h-screen max-w-md w-full relative bg-[#4D394B]`。
- **左侧主结构** 利用 `w-[88%]` 控制了实际紫底内容的绝对视觉主轴，采用 Flex 纵深流。
- **右侧截断面** 利用 `absolute left-[89%] top-11 bottom-0` 吸附在右侧，内部元素靠左对齐 (`pl-3`) 实现切割和遮罩，并在左边缘挂载深渊阴影 (`shadow-[0_0_25px_rgba(0,0,0,0.5)]`) 来突出两层之间的悬浮差距。
- 所有绝对定位功能图标（如在线绿点标记、翻页黑点器等）均按 1:1 利用 `position: absolute` 实现完美堆叠。

## 🎨 视觉规范

### 1. 自定义细节 (Design Tokens)

- **主体背景色**：暗紫，即特定的 Slack Aubergine 色系 `#4D394B`。
- **字体与强调色**：
  - Header 与用户粗体文字：#ffffff (纯白)
  - 弱文本与次级内容：#A693A6 (透气紫灰)
  - 状态呼吸灯：#2BAC76 (Slack 的标准活力绿)，包含对应同色系的底框遮罩实现呼吸空隙。
- **图标细节**：采用 `lucide-react` 图标集补充原图的标准 UI 功能件，包括 `Edit`(编辑)、`Navigation`(iOS信号导航)、`Signal` 以及截断面的 `User`, `PlugZap`, `AtSign`。

### 2. 组件状态定义

抽屉完全展开状态 (Drawer Open)，右侧页面完全右移。纯展示用无额外点击动效状态（所有可点击列表与编辑图标视觉静止）。
