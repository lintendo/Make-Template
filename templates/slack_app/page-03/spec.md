# Slack Channel System Messages

## 📋 概述

### 1. 核心业务与场景定位

这是 Slack APP 原型中的普通频道页 (`#designtools`)，主要展示一个纯系统事件（用户加入频道）的滚屏流水账日志视图。此页面属于静态浏览状态，即无激活键盘或复杂浮窗的底层消息页面。

### 2. 用户目标与关键任务

- **事件回溯**：查看各时间节点加入 `#designtools` 频道的人员记录。
- **联系人关联**：能够分辨 `@[UserName]` 的@提及标记样式。
- **时间界限感知**：利用带横线的日期分隔符明确日志的时序分段（如 May 16th, Yesterday, Today）。

## 🧩 内容规划

### 1. 信息架构与模块划分

页面由上至下划分为四大区域：

1. **iOS 状态栏**：时间 (7:53)，信号指示，4G标识，电池状态（满电）。所有图标为黑色，适应浅色白底模式。
2. **Slack 顶部导航栏**：
   - 左侧展示标准的彩色 Slack Hashtag (`#`) 图标，以及粗体黑色的频道名称 `designtools` 和下拉展开箭头。
   - 右侧包含：带独立免打扰标志 `zZ` 的 Bell（通知），搜索放大镜，以及竖向三点菜单。
3. **内容展示区 (聊天历史)**：
   - **日期分割线**：利用 `May 16th`, `May 19th` 等文案加紧贴右侧水平灰线的设计，切割不同的消息块单元。
   - **系统消息体**：包含随机几何图案头像（非真人），用户的粗黑名称与弱灰时间戳。消息体格式规整，统一为 `"joined #designtools ..."` 的固定句式。
   - **提及高亮 (Mentions)**：针对系统消息内的 `@` 成员标签做了基于背景色填涂与蓝色字体的圆角块高亮以区分普通文案。
4. **输入与工具栏 (底座)**：
   - 第一行：带有扩大（全屏）图标的 "Message #designtools" 暗灰提示条。
   - 第二行（功能按钮）：快捷键 `@`, `/`, `附件`(曲别针)，以及靠右侧的 `图片` 上传与静态的未激活 `Send` 按钮。
   - 最底部包含 iOS 的黑色 Home 防误触指示条。

### 2. 数据与内容来源

所有内容均为 Hardcoded Mock Data。包含六个不同时间段的人物进群记录。头像是专门为还原 Slack Identicon (几何随机生成色块) 而徒手利用 SVG 拟真出来的图形。提及 (`@domophysics` 等) 使用自定义 CSS utility 类控制视觉样式。

## 📐 布局与结构

- **整体架构**：外层包装使用 flex-col 铺满全屏高度 (h-screen)，限制最大移动端宽度 (`max-w-md`) 并设置居中及防选(`select-none`)。
- **中间内容流转**：通过 `flex-1 overflow-y-auto` 保留大量垂直空间以支持滚动流。日期分割线直接复用 flex 弹性布局将横线撑满剩余宽度 `flex-1 h-[1px]`。

## 🎨 视觉规范

### 1. 主题来源与使用情况

全程运用基于 Tailwind CSS 给定的基础类原子样式组合，并且针对提及文本的高亮颜色进行了 `style.css` 自定义声明。

### 2. 自定义细节 (Design Tokens)

- **字体排印与色值**：
  - 日期分隔符：15px 黑体。
  - 用户名称：15px 加粗黑字 (`text-black`)。
  - 弱化语与时间戳：`#616061` 和 `#9CA3AF`（部分采用 `text-gray-400`）。
  - Mentions 高亮样式：背景浅蓝 (`#E8F5FB`)，字体深蓝 (`#0B4C8C`)。
- **Identicon SVG 设计**：复原了 Slack 默认未配置头像时基于红 `#E01E5A`、黄 `#ECB22E`、蓝 `#36C5F0`、绿 `#2EB67D` 四原色组合切分的纯几何体拼接视觉特征。

### 3. 组件状态定义

纯浏览态 (Idle State)。所有的发送与表单元素均为非聚焦 (unfocused)，键盘收起，不涉及任何可交互变换阶段。
