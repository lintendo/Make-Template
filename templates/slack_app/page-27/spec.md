# Slackbot Standard Chat Feed (page-27)

## Design Overview
This node constitutes a precise `1:1` visual migration of the "Slackbot Direct Message" layout. 

## Component Implementations
- **Navigation Stack Drop-in**:
   - `SlackBrandLogo` manually codified leveraging interlocking path parameters reproducing the unified core hash emblem.
   - `BellSnooze` implements independent path generation fusing the standard `lucide-react` notification bell with an abstract `Zz` sleeping token override.
- **Top Greeting Container**:
   - Anchored without primary borders save for the terminal `#E5E5EA` `1px` divisor.
   - Expanded version of `SlackbotAvatar` invoked leveraging vector scale parameters mapped specifically to native constraints.
- **Message Feed Layout**:
   - High fidelity font scaling mappings for sender names and standard trailing gray timeline tokens. Text uses `15.5px` definitions mapping perfectly against standard iOS WebKit scales.
- **Floating Input Composer**:
   - Rendered using absolute padding layouts combined with an outline action mechanism (rendering the "Send" container as disabled line-art).
