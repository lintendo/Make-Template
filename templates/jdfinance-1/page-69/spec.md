# 白条权益-新手引导弹层 (2/4 步) (page-74) 规格文档

## 视图说明
`page-74` 为“白条权益”下方的全屏 Overlay 引导进行至第 2 步时的界面截取。
它和 `page-73`（第 3 步）处在同一条物理走廊上。但本视图中底部弹窗焦点是 `月月领`（而不是金融特权），并且展现了 1x3 的横向优惠列阵组合。工程复刻上的困难点与亮点在于多重微型徽标（Luckin 咖啡与 QQ 音乐会员）的无图重构，以及那个具有破风视效倾斜斜切角的“闪促神券”按钮（Split Price Button）。

## 零资源前端重构技术概览

### 1. Brand Logo CSS Mockups (跨域品牌的矢量仿造)
如果不采用 WebP 切图，就需要直接用 CSS 和系统 Unicode 在小方块中模拟强标识：
- **瑞幸咖啡 (.mock-luckin)**: 依靠 `linear-gradient(135deg, #4364c7, #243b92)` 等瑞幸蓝为基底建立正方形盒子，在中央嵌入一个巨大的 Unicode 驯鹿头/小鹿表情 🦌。在特定的字号下，蓝底+鹿的视觉映射会让用户立刻关联到品牌 IP。
- **QQ音乐超级会员 (.mock-qqmusic)**: 黑卡基底 `linear-gradient(135deg, #333, #111)` 奠基，右下角嵌入超大号、低不透明度的水印文字 `V`。在卡面上方漂浮着 "QQ音乐" 白字与一个底色为金黄色条纹 `.qq-badge` 的独立徽标，甚至在徽章内部利用小号 SVG 绘制了钻石钻面，以此重构出奢华黑胶白金会员卡的物理质感。

### 2. Multi-state Complex DOM Button (闪促斜切按钮系统)
“瑞幸”和“QQ”的购买按钮 `¥0.01 / 领` 具备一套视觉冲突极其抢眼的分割线排雷（Split Design）：
- 利用 `.btn-split-price` 容器封装，并通过 `flex` 将按钮分成 `left` 与 `right` 俩部分。
- **斜切角实现**: 核心在右侧的 `领` 字白块。为了实现向左上角的锐利冲刺切割线，并不需要用伪元素构建多余的三维体积，只需要在其自身的 css 添加遮罩滤镜：`clip-path: polygon(25% 0, 100% 0, 100% 100%, 0 100%)`。这个 CSS polygon 函数切掉了白色方块左上角的四分之一物理空间，露出了下方的红底容器，完美形成闪电折痕。
- **二级信息压缩**: 按钮内部署了三段异构超密文字（巨型黑体 0.01 撞击下划线特小的“已补¥99”），依靠 `scale(0.85)` 与 `transform-origin` 控制，强行把文字塞进 `10px` 行高，完成极其精致的电商级营销组件。
