# 签到领现金大厅页 (page-41) 规格文档

## 视图说明
`page-41` 属于应用内部的“每日签到打卡大厅”。
作为一个高度侧重 Gamification (游戏化) 和运营转化的独立模块，该页面的视觉张力极高，使用了异常明亮的红色调、海量的金币元素以及各种高对比度按钮。由于该页高度依赖插画，所以在无素材重构过程中面临了巨大的前端绘图挑战。

## 零资源前端重构技术概览

### 1. CSS 几何绘图体系：吉祥物重构
在签到卡片右上角，有一只探出头来的 3D 吉祥物小狗（京东金融的 Joy）。在严禁导入外部图床素材的要求下，我将其转化为纯几何重构：
我抽离开发了单独的 `.dog-mascot-hull` 及其内联组件（`.dog-head`, `.dog-ear-l/r`, `.dog-snout` 等）。将 5 个不同的白色/红色 `border-radius: 50%` CSS 圆球与半圆结构拼接绝对定位在右上角的卡牌挂角上。成功在 0kb 图片开销的状况下构建了一只具有相同神貌的运营装扮标识。

### 2. 现金抽奖巨核渲染 (Central Megacard Grid)
作为全页面重心，带有 **9.53** 大数字看板的部分经过了多道工艺封装。
- **微网格背景 (`.grid-lines-bg`)**: 数字背板那种“账本”一样的浅灰色水平线谱，通过 `repeating-linear-gradient` 直接生成覆盖其后。
- **深色下凹阴影 (`.lottery-card-inner`)**: 中心 6 宫格卡片不仅要保证白-绿的拼接结构，还要体现出立体的凹槽质感（仿佛是按压开关），这一步通过注入内部弥散投影 `box-shadow: inset 0px -4px 8px -4px rgba(12...` 实现物理模拟。
- **流光渲染大圆角按钮 (`.lottery-btn-glow`)**: 底部“立即抽提现”主行动召唤 (CTA) 抛弃了原生红底配置，叠加了复杂的内发光渐变与红色外阴影，这模拟了游戏 UI 中的晶体高光按钮光泽。

### 3. 多层绝对悬浮系统 (Z-Index Overlays)
- **待提现印章水印**：采用一个黄色边框配合 `.watermark-stamp` 类，不仅设置了近乎透明的字号（opacity: 0.35），并且精确植入了 `-15deg` 的左偏扭转，在第一张业务卡中达成了完美的物理盖章质感。
- **顶部横向进度轴**：利用了一个 `bg-gradient-to-r from-green via-gray to-gray` 的横线轨作为底架。两端绑定了白点与带钩绿点。确保逻辑解耦、完美水平相切。
