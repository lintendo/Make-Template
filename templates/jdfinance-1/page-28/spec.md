# 小病应急 (page-29) 规格文档

## 视图说明
本页面是在“家庭应急金”板块业务矩阵下的游戏化体验模块：“天天攒保障”。
这是一个经典的互联网积分/额度任务页面。页面高度依赖悬浮金币、光影和打卡游戏化逻辑，用于吸引日常日活并派发金融体验额度。

## 布局体系与动效重构思路

1. **游戏化穹顶光照体系 (Golden Ambient Lighting)**
   - 为了重构类似于“金库/阳光”的深空光感，并且抛弃庞大的 JPG 图片，我在 `style.css` 部署了重型背景 `.gold-hero-bg`。它使用基于 `circle at 50% -20%` 坐标的 `radial-gradient` 极化处理，完美还原了顶部强曝光、四周自然衰减柔和黄晕的视觉。

2. **三维浮动金币矩阵 (Volumetric Interactive Coin Orbits)**
   - 页面分布着 6 颗悬空的金币（例如: 18.00 / 188.00）。
   - **零图片重构**: 金币 `.golden-coin` 使用了双层高能态渲染：外环包含内阴影和半透明硬边界，内环则是对角线渐变。二者合一，不仅无需加载任何图片，还在任何缩放尺寸下均保持矢量清晰度。
   - **交互赋能**: 为每个金币赋予了不同的 `@keyframes` `bounce` 参数和相位差延迟（如 `0.2s`, `0.8s` delay），当你在页面悬停时，所有金币都在进行随机呼吸般的上下浮动漂移，1:1 还原了极高的“待提取/待收取”动态激励感。

3. **巨型护盾水印 (Massive SVG Shield Watermark)**
   - 核心数字 `0` 背后的淡淡护盾使用了纯内生 `base64` SVG 作为背景水印挂载于类的 `background-image` 中。其线条使用了 `<linearGradient>` 半衰减光效。依然秉持全页面 0 外挂图片准则。

4. **抽屉式卡片层叠与指向标 (Pointer Tabs Layering)**
   - 为了表现“1种重疾保障”与“100种”两页签的物理压制感，左侧高亮 Tab 借助纯 CSS `.tab-pointer-down::after` 给模块插上了一个尖锐的向下红三角，死死咬合住下方的白色内容基座卡套。
   - “做任务” 标语中的“提升保额”使用了 WebKit 级的空心字内晕染重构 `<span style="WebkitTextStroke: '0.5px white'">` 提供硬朗漫画式的边缘。
