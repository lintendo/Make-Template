# 数据资产与走势详单页 (page-44) 规格文档

## 视图说明
`page-44` 是用户的“收益详情与走势图表面板”。
页面结构上从重度营销感剥离，回归到了金融软件特有的严谨数据看板形态：大面积应用白色底板、去除了所有圆角阴影色块，排版极其简洁，使用了较小的灰色标注字体和突出的翠绿色（正收益）数值反馈。

## 零资源前端重构技术概览

### 1. 复杂流体数据网格 (Flow Grid & Column System)
资产拆解卡片采用了三列并行的栅格系统 (`grid-cols-3`)。
为了严密对称且不干扰视线，列分割并没有拉伸边框线，而是给每一列的主标题下方植入了一个极细微微调过的标识划线 (`w-[14px] h-[3px] rounded-full`)。分别辅配以金黄、亮橙与赤红色（`#d97706`, `#ea580c`, `#ef4444`），作为资产配置防御性水平（灵活 -> 稳健 -> 多赚）的视觉指引锚标。

### 2. 混合式渲染结构标靶图表 (SVG + HTML Hybrid Chart Rendering)
图表面板并没有外包给哪怕是 `ECharts` 这类的重型前端库，一切皆基于项目的极限零堆砌（Zero Asset / Zero Dependency）开发规范强行手搓实现：
- **轴线架构**：使用通过 35px 步进（Step Amount）排布的 5 个水平 Flex Box 作为精准定点 Y 轴标尺线（`10.00`至`-9.00`），保证跨分辨率绝对一致渲染。
- **SVG 数据态填色**：基于当前标出的 0.50 利润线（落在正中央轨道上方），我在该水平坐标内通过 `<svg preserveAspectRatio="none">` 部署了一个宽度 `100%`，并在内部封装 `linearGradient` 向下渗透的淡米黄色底液填充，物理表现出资金留存的积压感。
- **浮空工具提示节点 (Tooltip DOM Injection)**：在对应轨道的结束点附加了一个带下坠缺角的物理气泡 `.popup-marker`，且保证其位于 Z轴顶层避免遭数据轴网格切断。

### 3. 数据缺省状态抽象雕刻 (Empty State Abstract CSS Arts)
卡片最下沿的“收益明细”明显处于无数据的默认状态，因为原图展现了一处柔和插画。
为了在不切入图形对象的同时保持视觉重心的和谐，我开发了体系化 CSS 图纸 `.empty-state-card` 和 `.floating-diamond`。通过组合背景虚化的米色盒子、内置切割了多重几何边界的仿窗口 `clip-path`、以及外部被旋转抛离的菱形星体点缀……它们以绝对静默（opacity: 0.9）的方式坐落在盒子底部边缘充当优雅的空缺占位器。
