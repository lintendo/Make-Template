# 全部服务 (page-19) 规格文档

## 视图说明
本页面为京东金融APP全局工具大聚合入口 (App Hub/App Drawer)。该页面的核心职能是对应用内所有的二级/三级模块进行扁平化铺设。
从视觉上，本页完全被“模块化白底卡片”与“图标群列”填充，是极高密度的组件平铺页。

## 布局结构深度解析

1. **统一容器组件 (`ServiceItem`)**
   - 鉴于此页面拥有高达 **20+ 个图标映射**，我设计了基础的 `<ServiceItem />` React 组件闭包来统筹网格模型。
   - 规定图标槽位尺寸 (`width/height: 36px`)，强绑定行下文本居中与字号 (`12px`)。
   - 提供可选注入的 `badge` Node 属性，方便对于如“车备 / 养猪猪”这些具备极细粒度“角标”、“促销贴片”的业务模块进行精准绝对定位覆盖挂载。

2. **零资产纯 DOM 图标群 (Pure HTML/CSS Emoticon Engineering)**
   - 这是本页工程构建中的**最重难点所在**。所有的 22 枚微小图标徽章**全部基于前端原生 DOM 渲染！没有请求一张切图！**这保障了原型的极端轻量化与加载零延迟。
   - 拆解核心重制技术：
      - *几何裁切* (`clip-path`): 在【保险】区块，使用 `#5382ff` 背景蓝结合 `polygon` 多边形算法直接切除了标准的六边形防护盾牌底座，再将文字“车”内嵌。
      - *微缩遮罩层级迭起* (`z-index`): 在【互动福利】如“掼蛋”（扑克牌图标），利用两层长宽比特殊的白底容器，使用 `rotate-12` 及 `border` 重组出错落的扑克牌物理散落态。
      - *微图元拼接* (`Miniature SVG`): 在【养猪猪】里，绘制了主粉色圆形底，加上了一个微微倾斜带特定圆缺角的红褐色微圆圈用作“猪鼻子”，再附带 `[包邮]` 红底白字胶囊遮盖右下角边界。
      - *光影质感*: 所有圆环基础均被强制赋予了符合现实应用材质感的光源渐变：从左上角起（如 `from-[#fc7270] to-[#f45c5c]`）延伸至右下角的同色系递进，加上 `shadow-sm`。

3. **模块切片间距**
   - 分配为主业务的“保险保障”、“生活服务”、“互动福利”三大卡块。采用标准圆角率 `rounded-xl` 配以非常空灵的高级投影 `shadow-[0_2px_10px_rgba(0,0,0,0.015)]` 避免卡片对视线的过分抢夺。

4. **底部幽灵水印 (Ghost Logo Stamp)**
   - 对于瀑布流模块网格的封底，采用大幅度 `opacity-30` 层深，配置了黑线模拟的微缩京东 Joy 头像盒标识。配合横线占位符与“简单·快捷·安全”的文字组合锁死视觉结尾。
