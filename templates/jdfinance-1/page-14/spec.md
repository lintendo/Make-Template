# 理财频道首页 (page-14) 规格文档

## 视图说明
本页面为京东金融APP全局顶级 Tab “理财” 的频道首页视图。页面承载了公域流量分发、全局基础导航模块、用户资产安全信任背书展示，以及基于不同风险偏好（稳健/进取/高收益）的个性化资产推介。

## 布局结构深度解析

1. **顶级背景与全局头部 (Header & Background)**
   - **底纹映射**: 基于 `finance-header` 线性渐变搭配利用径向渐变模拟的 `guilloche-pattern` (防伪纽索暗纹)，烘托金融应用的严肃性及高级安全质感。
   - **交互顶栏**: 
      - 右侧保留标准的主机位全局搜索与带有新消息提示（小红点）的客服耳机 Icon。
      - 左侧采用强业务识别的“数智服务机器人”头像占位（CSS 原生重构），并且链接着“弹幕气泡框”形态的搜索框，输出实时新闻条目（例：“三大指数集体收涨...”），建立智能化对话感官。

2. **信任状横幅 (Trust Guarantee Banner)**
   - 处于强曝光区间的 `guarantee-banner` 粉底图块。
   - 此模块属于品牌信任资产输出。核心强调“为3亿+用户提供金融服务”及监管合规“证监会批准”、“民生银行资金监管”。
   - 右侧植入了高度风格化的全原生 CSS 及 SVG 路径构建的“金星/金杯”复合大奖图形。免除了纯图片载入的体积与适配烦恼，呈现高还原的光影叠加与磨砂混合边缘(`mix-blend-overlay`)。

3. **核心金刚区通道 (Primary Navigation Grid)**
   - “小金库/基金/稳健理财/股票/黄金” 五大流量矩阵。
   - 所有五枚图标均摒弃外部图像，基于 CSS 原生渐变构建了强隐喻的几何图形：
      - *小金库*: 笑脸橙红色锁包。
      - *基金*: 红色方格叠加大盘上升动向标。
      - *短债/稳健*: 蓝色复合圆盘+内核护盾孔。
      - *股票*: 红涨趋势的 K 线蜡烛图模型。
      - *黄金*: 错落推挤的金条微缩阵列。

4. **个性化权益提取区 (New User Benefit / Holdings extraction)**
   - **新人福利面版**: 大面积白底卡片强打 `6.0mg` 等核心物理计量资产（黄金），通过巨宽的红色大按钮 `立即提现` 实现转化截留。

5. **左右结构资讯/标的区 (Split Opportunity Cards)**
   - **左侧行业气象 (`债市晴雨`)**: 右上角原生 CSS 切割出光晕效果巨大的 `太阳` 天气图标；底部排列了类似于原生系统天气 APP 的五日气象横向微缩看板，清晰传达出当前板块“连涨（向好）”的市场晴雨表。
   - **右侧定点推介 (`自选机会`)**: 配置有巨幅红色夸张化数据收益 `+58.24%`，且底部配置了利用 SVG 强行勾勒的红色陡峰爬升 `Sparkline`（迷你有向图），用以增强该产品的短期动能直觉。

6. **全局 APP Tab 容器 (Global Bottom Navigation)**
   - 构建了 5-Tab 全端通用结构（首页/理财/省钱/社区/我的）。
   - 当前在 `理财` 并发了激活样式态（高光红/图标替换为红色存钱罐/钱包映射）。
   - 同时复刻跨越 Tab 隔离的新模块宣发（“社区”上方的异形悬浮气泡 `提现`），精确对齐全局通知视觉体系。
