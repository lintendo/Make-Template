# 个人中心 - 我的 (page-31) 规格文档

## 视图说明
本页面是 `page-30` 的镜像状态（完全置顶状态）。
它揭示了“我的”界面真正的头部核心：用户个人信息的展示区，以及背景中巨大且极具品牌感的几何水印设计。页面主体与 `page-30` 保持绝对对齐计算，但在 `保单与金额` 卡片的数据文案映射上发生了截留改变。

## 架构拆解与重构复原

1. **巨型几何阵列水印 (Hexagon Array Watermark)**
   - 为了复原用户头像右侧背景中极其庞然、由多级细线构成的“晶体护盾/六角形”结构（这是典型的大厂金融安全潜意识图案），我开发了 `.hexagon-watermark-bg` 独立图层。
   - 这并非一张贴图，而是 `data:image/svg+xml` 原生描边多边形注入！利用了 `<path d="M50 0L93.3013 25V75L50 100L6.69873 75V25L50 0Z"...>` 计算出了标准的六边形轨道，再配合底部的 `radial-gradient` 晕染背景，实现了无尽清晰、随时拉伸的完美大盘底色。

2. **用户高定虚构化身 (User CSS Avatar)**
   - 没有默认头像图片可用，这正是极限 UI 开发的有趣之处。我在 `.user-avatar-bg` 这一基座之上，利用 `#eb8c39` (肩颈) 和 `#32201e` (发型) 两块基本的绝对定位 `div`，切角 `rotate-12` 构建了一个低像素化的男性用户剪影头像。配上极其考究的内陷光影，使之非常接近系统占位符的感觉！

3. **微型文案重构 (Copywriting Adjustments)**
   - 虽然底部服务区组建和 `page-30` 是同一个单页面实例，但在该设计稿中，“保单与金额”主卡片的字段被降级简化了：从极度专业的 `保单总计 (含失效) / 预计来年保费` 退化至了极简的 `我的保单 / 保障金`。代码中对其进行了精准分离。 

通过这套逻辑，界面获得了完整的物理完整性，没有任何一寸死角留给模糊的 PNG 切图。
