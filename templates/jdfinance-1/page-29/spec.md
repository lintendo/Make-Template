# 个人中心 - 我的 (page-30) 规格文档

## 视图说明
本页面展现了京东保（保险频道）的“我的”个人中心总览大盘大盘。
其承接了对用户资产、保单以及个性化售后（理赔/客服）统管的作用。视觉上相比前面深橙/深金的高强度导购营销画面，回归了极简白色的“中后台管理”冷调感受，并通过模块化的圆角大卡片对功能进行解耦。

## 架构拆解与重构复原

1. **红雾流光顶栏与通顶错位卡 (Ambient Flare & Overlay Card)**
   - 为了重塑顶部的“微红晕染”（模拟品牌色的光感反射），我在全屏背景 `.profile-hero-bg` 植入了偏移圆心的 `radial-gradient` (中心从屏幕右上角 `80% 0%` 开始)。
   - 包含着“0 份”与“58 元”的数据概览卡直接插入这一块红晕背景下方，构建物理透视的前后景。

2. **多态保障总控中枢 (Insurance Hub Grid)**
   - 中部最大的全白卡片统管了所有的核心主营业务：“人身保障、家财车宠、储蓄 | 教育”。
   - 动态横向 Tab 激活后，在“人身保障”正下方部署了纯 CSS 创建的小水滴下巴指示器 (`.tab-safety-indicator::after`)，这比常规的下划线更带有一种“下拉收纳”的隐喻。
   - 内部包含的一个深层嵌套蓝色推荐卡套用了 `.abstract-medical-bg` 冷色渐变，辅以一个无图片的 SVG 老年医患场景轮廓重叠层，再次避免了使用 `.JPG` 图片，把高保真和 0 体积加载的强迫症原则进行到底。

3. **微量体元素精加工 (Micro-Branding Accents)**
   - 所有的右向箭头、特俗的小京灵专属红点呼吸灯 (`animate-pulse`)，甚至是理赔板块的彩色文本框内的实心桃心（安心赔 ❤️）全为纯净的 SVG/CSS。
   - 页脚拥有了最为复杂的叠拼认证：“JDA京东保 / 平台运营许可 / 津ICP备案号”。甚至连最底部的京东金融 Mascot（小狗/机械人徽标）与“简单·快捷·安全”两边带有横线的破界对齐文字 (`before:` 与 `after:` 伪类伪造分割线)，全部被还原。
