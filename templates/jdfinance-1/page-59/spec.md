# 京东小金库全景大盘页 (page-64) 规格文档

## 视图说明
`page-64` 是真实且清晰的京东金融小金库总览大盘 (Asset Main Hub)。
其最引人瞩目的是覆盖了大半个屏幕的主题色营销背景——呈现从淡雅到浓艳深红的高级热烈过渡，内部还镶嵌了各类金币高光与促销区块。该页面综合了核心资产数据展板、红包促活抓手、金刚区导航矩阵与重点产品推销流四个层级逻辑。

## 零资源前端重构技术概览

### 1. 超级圆弧网格与氛围光晕 (Curved Hero Backdrop & Halos)
完全没有使用哪怕1KB的背景贴片，全部通过 CSS `linear-gradient` 重塑了这张大红底：
- **圆弧切割 (`.hero-curved-wrapper`)**: 使用了超出屏幕宽度的绝对定位盒 `left: -20%; width: 140%;` 并切取 `border-bottom-radius: 50% 10%` 获得了一条顺滑的贝塞尔级穹顶底线。
- **金币光晕投影 (`.gold-flare`)**: 在穹顶底部游走的金色金币效应对视觉至关重要。我通过注射三个重叠交错的椭圆形放射状渐变 `radial-gradient(ellipse, #ffdf8c 0%, transparent 60%)` 模拟了背景深处的反光，在完全没有几何体图片的情况下完成了光斑伪造。

### 2. 多重嵌套卡片引擎 (Hierarchical Glass Cards)
- **破防飞出标签 (`.tag-right-overflow`)**: 主数据表右上角的必中返现标签通过 `absolute right-[-15px]` 打破了卡版的物理隔断，与后方背景结合构建了突出的透视层。
- **纯代码伪造刮刮乐 (`.scratch-ticket-box`)**: 中央 618 大促并非一张 Banner，而是利用 `mix-blend-overlay` 和 `box-shadow` 内发光嵌套模拟出的彩票材质。下方的“立即刮奖”按钮使用了绝对定位反向吞噬 (`mt-[-14px] z-10`) 横跨于彩票卡上，还原了截图中紧密的 UI 压边效果。

### 3. 多色矩阵图标库 (Grid Dual-Tone Icons)
复用了并进一步强化的 Dual-Tone (双语态线稿) 机制，使用主色 `#222` 与点缀色 `#ee8c5c` 重新绘制了“收益购”(包裹内嵌字)、红包、客服汽泡图等全新高复杂 SVG 图标集，避免了所有的外挂图片调用，真正实现了秒级加载的高频前端应用层。
