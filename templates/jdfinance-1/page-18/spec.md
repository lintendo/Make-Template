# 充值中心 (page-18) 规格文档

## 视图说明
本页面为 “充值中心” (Mobile Recharge Hub)。在此界面，用户管理充值记录、挂载代扣券，并被大量交叉销售的优惠逻辑（话费直减、金融开户送赠券）吸引和拦截。

## 布局结构与核心映射方案

1. **科技感充值头部体验 (`Hero & Phone Input`)**
    - 利用复合径向渐变 (`radial-gradient`) 拼装 `.recharge-hero-pattern`，模拟出类似金融卡的纹理脉络散开的视觉质感。
    - 使用极简玻璃拟态 (`bg-white/60 backdrop-blur-md`) 构建细粒度的功能按钮 ("我的号码" / "+号码管理")，使它们完全融合入背景而并不喧宾夺主。
    - 主号码使用极端字号 `text-[36px]` 结合窄字距压缩，最大化呈现数字的视觉分量。

2. **充话费功能盘 (`Recharge Grid Matrix`)**
    - 使用 CSS `grid-cols-3` 定义矩阵。分别管理了三种形态的状态反馈：
        - *默认纯净态*: (`50元`、`免单`) 使用中性色调铺底。
        - *直减引导激活态*: (`100元` 到 `300元`) 使用浅粉色背景配合红色的边框线与主副标题染色，实现用户视觉流的诱导。
        - *特殊促销挂件*: (`-￥30`) 单独配备了一枚脱离常规排版的悬浮式压角 “红包/彩漂券” 图标。我在这里纯粹利用了 CSS 几何形叠加并附带了 `transform -rotate-12`，彻底告别外挂图片。

3. **双栏拼合促销卡 (`Product & Lottery Bridge Item`)**
    - 在展示“购买商品赠送彩票券”的组合促销区块时，实现了完美的 `Flex` 一比二横向分割。
    - **桥接符号**: 两个区域中间悬挂了一枚使用绝对定位卡在交界线上的圆环十字符 (`+`)。
    - 左边单独规划了一张类刮刮乐风格的底纹；左上角附带有 CSS 绘制的折角 `[赠]` 标签，底部重构出体育彩票的 `DP/CP` 斜切红线 LOGO 特征。

4. **实体卡券列表 (`Ticket Ribbon Extractor`)**
    - 下方区域包含三枚横向并排溢出显示的独立任务卡券。
    - **CSS 物理切角实现**: 开发了高复用性的 `.ticket-cutout` 组件机制。
      - 我没有使用死板的图片或者沉重的 SVG `mask-image`，而是利用 CSS `::before` & `::after` 伪元素。
      - 在容器左右侧拉出宽/高 `10px` 的绝对定位白色正圆形圆点（一半置于容器外侧边界），配合底层底色对消掉卡券背景色，生成了 100% 清晰完美的**模拟开孔齿痕边缘**。
      - 这个解法极大增强了卡券模型的真实度与边缘渲染速度！
