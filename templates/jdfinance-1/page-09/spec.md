# 基金版块自选 (page-09) 规格文档

## 视图说明
本页面为京东金融 APP 内部相对独立的子频道：“基本面”及“基金投顾”下的“自选”大栏目。主要呈现用户未添加自选基金状态下的空状态（Empty State）面板及对应的品牌信任背书区域。

## 布局结构解析

1. **核心导航层 (Header Navigation)**
   - 包含常见的左侧返回区域，右侧则是提供了功能按键（检索 `Q` 图标及 新增 `+` 图标）。
   - **标题切换块**: 提供并排的大字号文本（“基金” 与 “投顾”）作为独立子应用的主层级切换指示。其中“投顾”以黑色加粗作为当前激活/强调状态。

2. **市场数据看板 (Sub-Header Statistics)**
   - 左侧展示了宏观交易大盘指数（如：`上证50`）及其点位与对应涨跌百分比（绿色字体的 `-0.15%`）。
   - 右侧则有一个微型的模块“发车”图标用于特定的投顾快捷跳转。

3. **类目筛选控制条 (Filtered Tab Bar)**
   - 两个长条型药丸过滤项：`全部`（浅红底红字 - active state）与 `我的持仓`（浅灰底灰字 - inactive state）。

4. **空状态引导区 (Empty State Display)**
   - **吉祥物占位图**: 基于 CSS 层叠样式实现了类似吉祥物白狗造型在气泡内的打哈欠状态展现。包括气泡内的三个聊天加载点与整体外发红光的高斯投影，无需加载独立外部位移资源。
   - **反馈文案及诱导按钮**: “暂无自选”，以及引注为红底白字的 `+ 添加自选` 圆角大号主行动按钮。

5. **底部信任背书区 (Footer Information Block)**
   - 提供了一个显眼的灰白色域 (`#f6f6f8`) 用于渲染金融安全感知。
   - **安全标** 布局了三个并列的辅助角标图案及灰字标签：“证监会批准”、“民生银行资金监管”、“200万资金安全险”。（图标均采用 `strokeWidth=1.5` 线框体SVG化构建）
   - 下方是合规声明长文及带京东金融小狗形态商标以及品牌暗号的灰色组合文字组。

6. **区域专属底导栏 (Fund Specific Bottom Nav)**
   - 提供五组专属于理财交易链路的页面路由。
   - 分别为 `基金`, `指数`, `全球投资`, `自选` 以及 `持仓`。
   - 对于本页而言，`自选` 被完全高亮激化展现（方框十字形加粗全黑渲染样式）。
