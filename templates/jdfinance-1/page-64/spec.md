# 金条借款核心首页 (page-69) 规格文档

## 视图说明
`page-69` 为京东金融 APP 内部的核心借贷首页 (“金条借款”主屏)。
它是引导最高用户转化率的关键入口页面。页面整体处于非常高级与细腻的金棕色调下，具备一个硕大的沉浸式光源穹顶背景、带复合标签的大额授信额度主力展示卡牌、插嵌式票券样式营销组件以及自带微交互和透明底纹推荐功能的底栏板块。

## 零资源前端重构技术概览

### 1. Hero Golden Lighting Mesh (环境光渲染背景罩)
无需导入一张高切图以避免包体积过大，原生的纯 CSS 滤镜与混合模型即可制作媲美 C4D 渲染的质感：
- 首先通过一个主屏幕尺寸容器 `.hero-golden-bg` 赋予线性折角渐变：`linear-gradient(175deg, #c36b1d 0%, #db843b 30%, #f4ede6 80%, #f6f7f9 100%)` 以实现天空向地面的漫反射衰减。
- 然后注入 `::before` 巨大的边缘光晕圈 (借助 700x500px 的 `border-radius: 50%` 圆环和 `box-shadow: inset 0 0...`)，模拟环境光在弧形轨道上的折射。
- 接着注入 `::after` 核心的高光聚焦斑 (借助 `radial-gradient` 且外围设置为透明的散射层)，让页面产生景深层效果。这种技术不但 0kb 流量开销，而且在大屏幕或者折叠屏手机上永远都能无损重绘。

### 2. Coupon Micro-Clipping (`.promo-inner-border`)
为了实现 "618 限时专享" 版块那种被剪刀从内角咔嚓剪去一小块变成圆形豁口的实体优惠层造型：
- 给 `.promo-inner-border` 设置正常背景并且利用 `::before` / `::after` 分散安置为绝对定位于左上和右下两个顶点（`-3px`负位移）的一个白色（或者背景色）纯色小圆圈。
- 这个小圆圈视觉上“遮蔽”或者“啃食”了矩形的尖角，从外界看过去这犹如真实的金融印刷票券冲压断口，非常高雅。

### 3. Native Navigation Bar Icon Set (原生下巴导航线型标)
最底部的黑白灰主导航 (借款、还款、我的)：通过纯 HTML CSS 重构几何拼装，而不是拉取外部文件，进一步将文件传输降低为 0：
- **借款**: 圆角设 50%，黑底，中心书写系统自带的英文字体无头 `Y` 或干脆加粗倾斜的 `¥`，完成重现。
- **还款/我的**: 使用 `border` 控制内外边距拼装长脚线条模拟出向外转账与微笑人脸 (`::after`/`::before` 注入边线和圆弧)，它们永远保证高对比度和丝滑的边缘采样。
