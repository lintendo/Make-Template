# 白金合伙人弹出层 (page-53) 规格文档

## 视图说明
`page-53` 展示的是“白金合伙人”活动页面触发关键任务转化时的 Bottom-sheet 引导操作弹窗。
设计在屏幕下方占据主位，同时利用了半透明阴影 (`dark-modal-context`) 遮蔽原本的主视觉背景。页面主体内包含了一组垂直时间轴（Stepper）用以展示任务进度（已完成/未完成），并附加一枚带有强烈互动暗示的红色动作主按钮。

## 零资源前端重构技术概览

### 1. Backdrop Context Mocking (伪造底层虚化环境)
为了达到完美的截屏 1:1 观感，我们并未将弹窗放在纯黑底色上。取而代之地，在底部 `z-0` 层渲染了两个受 `filter blur-md` 处理的绝对定位集合体（以此粗糙但极其有效地重现了 page-52 中白鹅和橙色精灵的高斯模糊影像），这为上层的遮罩叠加赋予了极富层次感的玻璃透视现象效果。

### 2. 水印图章抽象构造 (Abstract Inked Rubber Stamps)
在任务进度面板中，原版使用了印上去的 “已完成” 红色印章和 “未完成” 的灰色印章！
为了完全用 CSS 重塑这质感，我们构建了复合组件 `.css-stamp`。
- 将字体设置加粗拉宽并设定倾角 (`transform: translateY(-50%) rotate(-15deg)` / `-rotate...`)，以模拟随手盖章的角度。
- 借助 `border-radius: 50%` 和伪类 `::before`，生成具有细缝镂空的经典实木双环印章圈结构。
- 通过引入中心透明发散至边缘略微有颜色的 `radial-gradient` 增加墨水印泥由于压力不均产生的中心淡出纹理感，再辅以 `opacity: 0.8` 使其实际“印图”与底下文本稍具混合叠加特性。

### 3. 指引交互波纹引擎 (Ripple Call-To-Action Simulation)
- 在大红按钮的右侧不仅叠加了跨页通用图腾 `.css-hand-pointer` 伸指点击，而且在这个按压焦点位我们植入了 `.css-ripple-ring`。
- 调用原生 CSS 硬件加速 `@keyframes ripple`，持续施加 `transform: scale(2.5)` 到边界的同心圆动态扩散动画，为静止的前端页面赋予了强烈的点击引导动势流。
