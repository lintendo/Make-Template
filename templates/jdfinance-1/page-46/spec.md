# 新客专属理财 (page-47) 规格文档

## 视图说明
`page-47` 展示了向新用户推销专属高收益理财券的落地页。首要目的是制造强大的利益点（8.18% 的显眼收益数字），辅助操作进度条，以及底部的富媒体（视频）教学模块，试图构建用户点击购买的安全感。

## 零资源前端重构技术概览

### 1. 复杂 DOM 折纸带 (Origami Ribbon Flag)
左上角突破卡片盒模型的“权益1”红底标签采用了经典的 CSS 折纸错觉：
- `.ribbon-flag` 置于左侧负空间 `left: -6px; top: 16px`。
- 采用伪类 `::after` 生成一个 `border-width` 不等大小的正切三角形图块，上色为深红（暗角，`#a41c1c`），并定位贴合负空间的底部间隙。它模拟了丝带被从父级卡片下方塞入环绕而出的3D物理深度感，是高级 UI 常见的破框设计语言。

### 2. 流行波普特效渲染引擎 (Pop-Art Video Thumbnail Engine)
原视频的封面是由复杂漫画网点、剪贴画元素构成的波普艺术风（Pop Art）图片。为了在零物料体系中复绘该内容，我封装了 `.css-pop-art-video` 复合组件：
- **Halftone 纹理基底 (`.pop-art-bg-texture`)**：利用 `radial-gradient` 阵列组合透明间隙，并加上15度倾斜，精准模拟了漫画书底层的半调网点打印颗粒。
- **立体字幅 (`textShadow` Stack)**：主文字“证券漫游记”叠加了复杂的多维黑红错位文字阴影以实现字效外描边加下沉投影；同时其上方的对话框 “Let's go!” 装配了下缀边三角形来生成完整的实体气泡。
- **几何元素镶嵌 (`.pop-sunburst` & `.pop-checker-box`)**：通过 `repeating-conic-gradient` 扇形切割，加上双层线性渐变的赛车棋盘格绘制技术，硬编码输出了所有需要用到的矢量抽象图案！

### 3. 工作流状态步进器 (Stepper Tracking System)
利用三层叠加法则（Base Line > Absolute Points > SVG Icons）搭建进度条。并在“当前激活动作” (`.step-node-active`) 的外部通过 `::after` 追加了一个带透明度的红色淡出环特效以形成视觉焦点呼吸感。
