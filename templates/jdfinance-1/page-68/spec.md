# 白条权益-新手引导弹层 (3/4 步) (page-73) 规格文档

## 视图说明
`page-73` 承接 `page-72`，是引导全屏覆盖层进入了第 3 步（下一步(3/4)）。
页面中央指引光标、大鹅及文案依旧维持悬浮态，但最为关键的变化是：底部弹出的半屏幕 Modal 内容彻底变了——从之前的 4 宫格蓝粉色块 `白条服务`，变为了两张硕大、高精度渐变呈现的左右双排 `金融特权` 展示卡片。本页面的工程重点落在右侧 “小金库加息券” 及左侧 “理财权益卡” 顶部的 3D 悬浮微景观动画模拟。

## 零资源前端重构技术概览

### 1. Advanced Mock 3D Structures (悬浮微景观与 CSS 几何建模)
底层不依赖外链 Image，只凭借浏览器的 DOM 树描绘出这两种微景观台面：
- **理财权益卡 (Podium & Honor Card)**: 左侧卡牌中心包含一个金光闪闪的冠军铸造台（台基）。这是使用 `.podium-base` 的圆角化 (`border-radius: 50%`)，加上不同层级的金棕色线性渐变 `#ffdb85`, `#cba245`，再利用 `::after` 定位压扁形成切平式三维圆柱底座。而底座上的悬挂卡牌，则是通过带小阴影和 15 度旋转的黑金实心块，内嵌一颗大号 Emoji ⭐ 代替高模渲染资产。
- **小金库加息券 (Phone & Stonk Charts)**: 右边是一张涨幅惊人的金融报表握在手中。我们让一颗尺寸放大至 70px 的手掌字符 `🖐🏼` 作为托盘背景 `z-index: 10`。被手托举的暗夜幽蓝色外壳智能手机，则是简单一个带圆角、高宽比约束和粗边框的 `div`。手机内部注入带有向上走势图标的白色屏幕 `div` 构成。最后借助纯 CSS 撰写的 `@keyframes float` 慢速上下漂浮几粒金黄色圆形实心硬币，构成了一幅活灵活现的赚钱幻景。

### 2. State Transformation Continuity (引导流的连续性模拟)
在保持与 72 同样的主题下，为了映射正确的背景位移反馈：
底层假背景组件群的透明度进一步衰减（从上层渗透率为 0.15 降低为 0.2 以适应暗化需求），且外包裹父容器进行了 `transform: -translate-y() ` 偏移模拟——这使用户切实感觉到背景是存在并且可滑动的，即便这是一组轻量化剥离了逻辑树的幽灵节点。
