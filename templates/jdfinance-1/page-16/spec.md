# 京东黄金 - 新手引导界面 (page-16) 规格文档

## 视图说明
本页面为强业务引导层 (Onboarding / Guided Tour Overlay)。
它建立在 `page-15` (**京东黄金主页**) 的视觉底座之上，覆盖全局暗色蒙层透出特定节点（底部 `首页` Tab），并附加专门的提示灯框、射线指引与操作跳位工具，用于强塞核心业务定位给新准入用户。

## 布局结构与特殊交互实现拆解

1. **底层结构 (Frozen Background Layer)**
   - 彻底克隆了 `page-15` 的核心展示骨架，剥除了强交互性 `hover`/`active` 反馈，使其保持在“冻结”呈现状态。
   - 保留顶部华丽底纹 (`gold-hub-waves`), 金刚区, `今日关注` 新闻列表结构, 工具卡及 `积存金` 等模块卡片，为蒙层提供真实的上下文。

2. **全局视通蒙版 (SVG Masking Engine)**
   - **抛弃常规 CSS Hack**: 不使用堆砌 4 个不同维度的黑色 DIV 拼合，也不使用浏览器支持率不一致的 `css clip-path` 镂空。
   - **绝对稳固方案**:
     - 创建覆盖了 `fixed inset-0 z-[60]` 的标准 `SVG` 容器。
     - 借助 `<mask id="guideMask">` 定义透光规则（矩形 `rect` 铺白指代遮挡不穿透的部分，圆形 `circle` 涂黑指代挖出穿透的高亮盲孔）。
     - 通过精准挂载 `mask="url(#guideMask)"` 到整体遮罩填充态面上，完美实现在 `x: 12.5%, y: bottom 28px` 点位的标准圆形探洞，精准暴露下方的“首页”控制栏按钮区域。

3. **导航焦点强化层 (Tooltip & Highlighting Node)**
   - 被 SVG 洞口照亮的区域，在其对应的 `z-[70]` 层额外补全了一张纯白的“首页”徽标组，来保障全息遮罩可能带来的灰度污染不再干涉这一首发强光点。
   - 在高光点的垂直正上方（`bottom: 90px`）：
     - **排版定位器**: “黄金全部功能入口都在这” 采用了重阴影 (`drop-shadow`) 提供立体的无背景文本漂浮。两边散落绘制了带有光晕 (`glow`) 的黄白两星星 SVG。
     - **引线组件 (`Pointing Line`)**: 使用绝对定位的垂直一像素 `<line>` 并在线端包裹小点状 `<circle>` 进行空间联结映射，形成指向明确的逻辑桥梁。
     - **业务流转动作**: 配置深灰带微折射效果（`backdrop-blur`）的胶囊提示按钮，下方保留典型的橘红细框指示点，提示当前为流程的 `(1/4)` 阶段。

4. **逃逸节点 (Dismiss Action)**
   - 处于全局右上角顶级遮挡区间域，配置带有半透明纯白描边的“跳过” (`Skip`) 操控区域。
   - 赋予 `z-[70]` 以保证不被引导逻辑本身的元素所影响阻碍点击穿透。
