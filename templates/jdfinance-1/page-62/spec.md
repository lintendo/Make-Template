# 金条借款平台-借多久(还款计划)页 (page-67) 规格文档

## 视图说明
`page-67` 为京东金融“金条借款平台”的“借多久”（即期数选择与分期计算）弹窗。
页面被分为两个高清晰操作维度：一个横向滑动的期数选择器 (Horizontal Scroll Bar)，以及一个高度拟物化且视觉清爽的垂直时间轴还款明细列表 (Vertical Timeline View)。

## 零资源前端重构技术概览

### 1. 结构化期数拨片组件 (`.term-box`) 及其状态生成
顶部提供了“24个月”、“18个月”、“12个月”横滑选择卡片。所有状态与装饰物全由 CSS 构建：
- **微型角标 (`.term-tag`)**: 通过外置绝对定位并挂载 `linear-gradient(to right, #f6a666, #e76f28)` 实现“按期还专享”的高亮徽章，并通过 CSS `transform: scale(0.95)` 处理到极限小的像素不虚化表现。
- **高亮角带对勾 (`.term-box.active::after`)**: 这个右下角的特色对勾状态没有使用任何 SVG 底底图。而是在 `::after` 中使用 `clip-path: polygon(100% 0, 100% 100%, 0 100%)` 裁切出一个严格的直角三角形，再往其中注入对勾 SVG 图标。这样在不同的手机屏幕上缩放都不会产生像素锯齿。

### 2. Time-line 核心渲染引擎 (无资源垂直时间线)
占据屏幕最主要视觉的左侧由圆点和连续垂线构成的“时间树”:
- 对于每一行 `.timeline-item`，其自身通过 `::before` 向下延伸出一条左侧绝对定位于 `82.5px` 的浅灰色连接线 (`width: 1px`)，直到穿透它自己的盒子延展到了下方的项 (`bottom: -10px`)。
- 采用 `.timeline-item:last-child::before { display: none }` 切断最后一行（或是被卷入溢出边缘）的无限延伸线，保证了视觉闭环。
- “圆点结点” (`.timeline-dot`) 则叠加在垂线之上，形成完美的穿引视觉。彻底解耦了图片对于 UI 项数变化的限制。

### 3. 多列表头的渐隐过界 (Gradient Fade Tab Overlay)
时间轴顶部拥有 `第一单` 和灰化的 `第二单` 标签。背景应用了原生的 `linear-gradient(to right, #fff4ed 0%, #ffffff 80%)`：
这样第一单的表头将沐浴在淡橙色氛围中，又非常柔顺且不着痕迹地滑入白色的基底里，展现了深度的金融产品交互质感。
