# 优惠券页 (page-03) 规格文档

## 视图说明
本页面为京东金融的“优惠券”管理页面，聚合展示了用户的各类优惠券资源（平台好券、推荐好券等），采用了典型的卡券切角和分割线视觉风格。

## 布局结构

1. **顶部导航区 (Top Area)**
   - **导航栏**: 
     - 左侧返回箭头（`<`）。
     - 中间大标题“优惠券”。
     - 右侧有两个操作图标：“记录”和“兑换”。
   - **分类选项卡 (Tabs)**:
     - 包含“全部”、“新到”、“快过期”三种状态分类，选中的状态带有加粗及底部红线标识。
     - 右侧有“筛选”及下拉箭头入口。

2. **核心业务区 1: 平台好券 (Platform Coupons)**
   - 标题：“平台好券”。
   - **票券卡片 (Coupon Cards)**:
     - 使用白色背景，外加通过左/右伪元素构成的半圆形凹槽“切角”。
     - 卡片中部带有虚线分割，划分为“左侧核心价值区”和“右侧详细信息区”。
     - 左侧显示立减金额（大字号数字+小字号单位），并在左上角贴有来源标签（如：支付、财富）。
     - 右侧显示券名、有效期以及右边缘显眼的红色实体操作按钮（“去使用”）。带有特殊的角标说明（例如橙底白字的“快过期”标签）。

3. **核心业务区 2: 更多推荐好券 (Recommendations)**
   - 标题：“更多推荐好券”，右上方使用 SVG 配合绘制的立体狗头及金币红包动效占位符作为视觉点缀。
   - **推荐激活卡片 (Pink Cards)**:
     - 此处卡片背景呈现京东标志性的浅粉丝系底色 (`#fff6f6`)。
     - 左侧同样为主视觉金额。
     - 右侧展示“京东支付券”，按钮样式由实心红变成白底红边样式的动作指引（“去激活”）。

4. **底部水印区 (Footer)**
   - 使用了极低透明度的灰色展示“京东金融”带图标 logo，并跟随 Slogan “简单 · 快捷 · 安全”，附带左右细横线装饰。

## 视觉与交互规范
- 背景色统一采用京东通用的浅灰配色 `#f5f6f8` 以凸显白底/粉底卡片。
- “优惠券卡片切口” (Coupon Cutout) 的实现摒弃复杂的纯图片切图，使用相对更易复用和拓展的纯 CSS `pseudo-element` 渲染凹槽（背景取巧），保证自适应宽度的良好表现。 
- 主色调采用标志性的京东红 `#e1251b` / `#fe3840` 作为可交互元素、主要金额数值或提示元素的颜色。
