# 金库持仓看板页 (page-45) 规格文档

## 视图说明
`page-45` 代表“金库+持仓”的总体视图系统。主要由顶部的资产汇总、中部矩阵动作菜单区以及占了最大篇幅的底部空状态提示构成（用户暂无购买资产时所呈现）。
这不仅是一个财务数据展示页，里面还包含着复杂的优惠拉新横幅。

## 零资源前端重构技术概览

### 1. CSS 几何插图：Joy空状态狗头 (Empty State Mascot Reconstruction)
面对底部卡片中心巨大的原画图片占位，为了在无外界图像引用的局限下达到1:1体验，我在 CSS 层实施了基于绝对标尺系的抽象角色装配：
- **容器与裁切**：先使用 `mask-image` 和径向渐变切割出 `.empty-ticket-bg` 这张边缘带撕齿痕迹的粉色半透明票根框。
- **肢体与五官拼接**：在 `.dog-joy-minimal` 组件中拼合多达12个 `border-radius` 形变模块！包括带角度旋转的狗耳（`joy-ear-l/r`），由单侧透明边框勾勒出的微笑嘴唇（`joy-smile`），以及最绝妙的——通过强行压扁 `height` 加粗上边框打造出右眼正在 wink (眨眼) 的弧线状态 (`.joy-eye-r`)。由此用纯 DOM 解析渲染了一只充满生命力的金库吉祥物。

### 2. 线描矩阵构建 (Line-Art Iconography)
对于中部那五个应用工具（交易、明细、分析等），因为无法拉取现成 SVG sprite，所以选择了更加性能极限的 CSS DOM 生成战术：
依靠极简属性构建 `.icon-outline` 系类，例如公文包 `.icon-doc`，通过赋予一个带有一根 `::after` 伪类的 `border`，然后利用 `box-shadow` 大间距复制产生内页横向纹理。这些图标能够继承父文本元素的 `currentColor` 呈现矢量级的清晰度缩放特性且体积几乎为零。

### 3. 微型促销装模 (Micro-Promotion Componentry)
顶部资产卡右上侧搭载了“领新人福利”小红签，在其内部渲染时挂载了独立的红晕高光 (`.promo-red-glow`)。而在页面中部的一条优惠券提示栏里，“红包图标”同样通过两层伪类（底层红方块被 `border-radius` 变形后由 `::before` 压下红色四角星钻石体）组合堆出。在不导入任何图片的前提下保证了运营物料的吸睛率。
