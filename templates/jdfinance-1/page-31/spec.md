# 京东保主入口页 (page-32) 规格文档

## 视图说明
该页面是整个“京东保”（JD Insurance）业务域的总路由与首页展示端 (`#保险` 频道的主视图)。
和之前子分类页面不同，该主干道的布局结构更侧重于大开大合的入口分发、抽奖营销事件（Banner）与超大号强推特性的商品橱窗。

## 高阶前端交互重构机制

1. **破裂放射状巨型特惠广告横幅 (Starburst Promotional Banner)**
   - 对于画面中间最高亮的“车主限时抽7000现金”横幅，常规开发几乎 100% 依赖出正方切图去包裹文本。
   - 这次我启用了极限纯代码构建：
     1. 第一底色使用 `.promo-banner-bg` 创建正中心爆炸光环（从 `#ff6a4a` 到最深红）。
     2. 交叉叠加 `.burst-lines` 的条纹旋转交替逻辑构造背后的光芒射线阵列。
     3. 利用 React 映射生成十二重内镶嵌旋转了 `30deg` 的透明 SVG 放射线 `<polygon>`。
     4. 中心“黄金环”：`.golden-ring-wrapper`，通过两组正负相碰的内外 `radial-gradient` 与 `box-shadow` 打出了强烈的抛光金属质感。配合字体的内排版彻底实现了 `0` 资源的抽奖广告。

2. **全局工具矩阵标 (Dashboard Tool Matrix Indicators)**
   - "车险、带病医疗、储蓄险、意外险" 对应的图标极不规范且充满插画特征。我在重构中将其物理降维进入 CSS 特性：
     - 利用 `border-l-transparent` 和负空间拼图造就了“带病”头顶的折纸效果。
     - 利用 CSS 矩阵旋转 `rotate-6`，配合线段堆叠伪造了红黄相间的意外警惕符号图。
     - “车主俱乐部”则是在色块叠加处穿刺剪影 SVG 以构成负空间的矢量图案。

3. **双重置顶体系 (Dual Header Navigation Layers)**
   - 本视图包含了上方的业务域名（`京东保`）搭配悬浮系统 Mascot 的 Navigation。
   - 底部全局 `fixed` 横栏内挂接着系统级的“保险”图标（通过切角剪切技术手绘的坚固方形盾牌）。
   - 最后，上端包含有可水平游动的横向分类栏，实现了完全与原生平台对等的 1:1 三向维度滚动骨架。
