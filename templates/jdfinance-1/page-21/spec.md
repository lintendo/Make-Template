# 全部服务视图 - 顶部首屏 (page-21) 规格文档

## 视图说明
本页面为京东金融APP全局工具大聚合入口 (App Hub) 的**绝对置顶视窗**。
这是该系列流的顶部起源态 (`page-21` -> `page-20` -> `page-19`)。它不仅陈列了四大全新分区块（首页应用快捷栏、最近使用、白条服务、投资理财），同时截断衔接了下文的“信用借贷”入口。

## 布局架构与复用机制提取

1. **内联宏组件中心仓库 (Macro Icon Generator Hub)**
   - 由于在此页面上，“最近使用”卡片会混合抽取来自各个不同业务域的高频调用图标（例如抽调“生活服务”的京享租、抽取“信用借贷”的金条借款等），为了保持代码的高度收敛和 1:1 设计保真，我在 `index.tsx` 首部设计了**全局函数式的内部聚合宏** (`Icon*` 组件系列，如 `<IconJingXiangZu />`, `<IconJinTiaoJieKuan />`)。
   - 这使得复杂的 CSS 抽象构建逻辑得到了完美复用，实现了极强的代码模块独立化封装。

2. **核心首版特性与 CSS DOM 重演**
   - **首页应用 (Home Apps Mock Banner)**：这是此页面最特殊的单节点。它拥有高度集成的重叠式次级图标流（水平轴平铺缩放）。
     - *实现方案*: 没有使用图片贴图，而是直接在 `scale-[0.6]` 且子元素 `space-x-[-12px]` 的 Flex 容器中循环调用底层的宏图标组件，配合 `absolute right-0 ... bg-gradient-to-l` 透明渐变遮罩，完美复刻出“拥挤待编辑应用列表”的视觉截断效果。
   - **全新业务域（白条 & 投资理财）CSS 图元构建**: 
     - *京东信誉分*: 运用 `clip-path` 修改出一个带有平顶和收缩底部的纯 CSS 实体盾牌形状。并内嵌白条微块构造出微笑嘴角的图形。
     - *黄金*: 依靠绝对定位、双层 `z-index` 和 `box-shadow` 内阴影，完全用简单的细长 `div` 堆叠构建了立体金条组的视觉反馈。
     - *小金库*: 利用 `border-radius: 4px` 的微缩红盒子，配合上下半边框裁剪生成“钱包扣/锁”闭合形态元素。
     - *股票*: 构建了标准的 `display:flex` 排版，分别注入带有细定位红条与胖粗主体红块合成经典的 K 线实体图。

3. **视图对齐控制 (View Synchronization)**
   - “信用借贷”模块被严格定义成了 `h-[120px] overflow-hidden`，在不调整原有网格排版的基础架构上强力复刻了上滑切断一半容器视觉的真实滚屏姿态。
