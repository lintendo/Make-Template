# 信用借贷服务视图 (page-20) 规格文档

## 视图说明
本页面为京东金融APP全局工具大聚合入口 (App Hub) 的**顶部视窗锚点展示区**。
它是 `page-19` ("全部服务") 视图向上文滑动的衔接内容，补全了应用中权重最高的顶级变现渠道板块：**“信用借贷”**。

## 布局结构

1. **顶级权重卡片注入 (`信用借贷`)**
   - 处于整个聚合服务列表列阵的最顶端第一顺位。
   - 配置了八个超高净值业务入口图标，均采用暖色系（高亮橙/深赤红）与金融稳重色系（深邃蓝）相交替，视觉引导极强。

2. **原生 CSS 微缩图标组解构 (The Lending Icon Matrix)**
   - 贯彻全节点无图元（Zero-Asset）开发策略，本区域新增的 8 枚金融图标全部由原生 DOM 与 Tailwind CSS 生成。 
   - *金条借款*: 采用 `from-[#fca466] to-[#f68940]` 黄金渐变叠层。巧妙使用绝对定位叠加一个负向偏移并且拥有相同背景反转描边的正方形盒子来精准裁剪出“折角文档”效果。
   - *农户借钱*: 不使用笨重的 SVG 放射背景，直接向圆形内硬塞三个带透明度的细线 `div` 进行 `[0, 45, -45]` 的角度旋转交叉，快速构筑阳光/麦穗放射光晕效果。
   - *车抵贷*: 利用极其精细的微缩 `border-radius` 与微偏移块建立了一辆正面朝向的微缩卡车/轿车剪影，其正中通过极致 `text-[7px]` 嵌套了“贷”字印花。
   - *小微借款 / 企业金融*: 通过利用 `border-x / border-[transparent]` 搭建红色的实体屋顶、大厦透视外墙，完美还原商业借贷店铺与楼宇实景抽象图标。并在它们的右下角全部统一挂载绝对定位覆盖层（模拟带阴影的抛光金色铜币 `[¥]`），强化信贷心理暗示。

3. **下文衔接**
   - 第二至第四张卡片（`保险保障`、`生活服务`、被截断的 `互动福利`）完美继承自同一规范模板体系进行瀑布流衔接。为了精确卡控一比一的屏幕呈现位置，调整了 `由于上文加入板块带来的视差` 与 `特定互动卡片的容器截断 h-[180px] overflow-hidden`，保证完全重合参考。
