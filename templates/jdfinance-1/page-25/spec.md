# 家庭应急金 (page-26) 规格文档

## 视图说明
本页面为京东金融APP旗下关于疾病门诊综合商业险的导购矩阵主页：“家庭应急金”。
该页面的核心职能在于罗列各类年龄段、高保单金额的金融特药产品，通过极其醒目和紧凑的 “红色大字标价 + 排他性保障标签” 来促使高净值家庭做出长线安全投资。

## 架构体系与仿真重构

1. **虚拟家庭肖像算法构建 (Abstract CSS Avatar Gallery)**
   - 原始页面中左侧陈列有多张高度写实的各阶层家庭生活摄影照（如老年夫妇、青年三口等）。为了在 0 文件体积开销的原则下还原其视觉高定感，并且不使用粗糙的灰框占位符：
   - 创造性引入了组合的 `.mock-photo-*` 类！
   - 通过将深浅不一的肤色系线性渐变 (`linear-gradient`) 叠加上微透黑纱渐变 `from-black/10`，并在正中心打入一个全透明只剩边缘的 SVG 抽象人像，在缩放预览时实现了极其精致的 “虚化人像照片结构”。完美替代了图片资源。

2. **高紧凑度电商排版流 (High-Density Commerce Layout)**
   - 每个白底网格行内集成了 **5种以上不同层级的排版字号**:
     - `text-[15px]` 标题
     - `text-[12px]` 灰色次级信息
     - 底部锚定的超紧凑价格区：将 `¥` 符号与 `/月起` 固定为 `text-[12px] or [11px]`，中间数字放缩为 `text-[20px]`，使用标准的基线对齐结合强制负边距以完成严密的错位。
     - 最后尾随一个由 `scale-[0.85]` 强制物理压缩的高亮橙底赠送 Tag 标签。这种极致的空间压榨是金融级 APP 的标志性样式。

3. **双重叠加行动面板 (Dual Floating Controller)**
   - 突破性实现了页面底部带有悬空半透明层级的双 Action Bar 控制器：
     - **次层级 TabBar (`z-50, h-[54px]`)**: 固态吸附屏幕最底部，并绘制了 5 极细的线型黑客风 SVG Icon（如饼状图、急救箱储蓄罐）；
     - **核心级 CTA 结算条 (`z-40, bottom-[54px]`)**: 使用巨大的渐变高爆点投影 (`shadow-[0_-5px_15px_...]` 伴随 `.action-bar-gradient`) 在第二层形成立体感隔离。巨大的弧形分享按钮 + 闪瞎眼的纯红 “0元体验”，使得用户视觉动线被 100% 聚焦于体验触发口上。无切割断层，自适应所有设备的高度溢流。
