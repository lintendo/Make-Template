# 储备金保险 (page-25) 规格文档

## 视图说明
本页面为京东金融APP旗下针对理财与储备金保险打造的业务门户：“储备金保收益3%+”频道主页。
不同于前面以科技红或农业绿为主导的业务，该业务因为主打女性储蓄、养老与少儿教育金，采取了大量柔和的粉白系与爱心抽象物作为全景视觉系统。

## 布局架构与微元素复刻

1. **柔情理财视觉体系 (Pastel Geometry Engine)**
   - 抛弃生硬的背景图，直接利用 `.wealth-hero-bg` CSS 背景叠加深浅粉桃色渐变。
   - 在高空领域 (`top-[-30] right-[-40]`)，创造性使用了 SVG `<path>` 画笔手绘了几个无实体的半透明爱心与房屋模型（包含 `fill="#f8b6b6"` 等），以此来模拟截图中象征“家”的理财保护伞暗纹。这完全保留了页面对缩放的支持度，并降低了 100% 的网络开销。

2. **伪钞纹理字效底纹 (Ruled Accounting Lines)**
   - 在主位展示傲人的 `3.17%` 大字的区域背后，拥有传统账本里精细入微的红色横条连线格。
   - `style.css` 里定义的 `.ruled-lines-bg` 结合了 `repeating-linear-gradient`，每 9 个像素执行一次完美的 1px 红线绘制，形成了与原版图片里 1:1 的账本纹理效果，这极大的提高了金融产品严谨复古感表现。

3. **对话气泡拦截栏 (Bubble Tail Dialog)**
   - “抓住机会 立即上车” 的大型 CTA 按钮顶端携带了一个促销迫切感极强的微型气泡。
   - 我避免了写冗长行内 DOM，直接为其挂载 `.bubble-tail` CSS 伪元素，使用透明 CSS border 切割魔法完美切出一个锐利的小三角形向下箭头，使得 DOM 树更加干净轻薄。

4. **金融下划线高亮笔触 (.brush-underline)**
   - 标题的 “3%+” 的下环绕不再是普通的直线下划线。通过赋予 `rotate(-1deg)` 、特定宽高比以及偏移的 `::after` 构造，它看起来就像一根人工划上去的高亮马克笔。

5. **实体储蓄罐 Icon 绘制**
   - 底部深渊级 TabBar 中的主激活态按钮使用了黑色的物理小金猪储蓄罐作为 Icon。我提取绘制了相应的 `<path>` 路径实现矢量显示，完美对应 `025-pic` 中的细节设计。
