# 理财精选好品页 (page-58) 规格文档

## 视图说明
`page-58` 是京东金融典型的“理财/权益 Hub”黑金风格聚合页。
页面总体背景采用深邃的曜石黑底色 (`#0b0806`)。核心由三大模块构成：
1. **十一宫格矩阵 (Privilege Grid)**：密集排列的系统级特权应用入口，采用 4 列布局和高度统一的深色图文组件。
2. **分类导航流 (Category Tabs)**：支持横向滑动的浅层锚点导航（如：活钱理财、稳健理财等）。
3. **主推爆款卡片 (Hero Product Card)**：占据视觉重心的纯白拟物卡牌，顶部带有极为特殊的“倒凸起/向上翻阅”实体标签，内部呈现主生理财产品的核心历史年化率。

## 零资源前端重构技术概览

### 1. 拟物化文件夹凸起标签 (CSS Upward Tab Cutout)
本页最具挑战的部分是白色主推卡片正上方的「灵活取用」橙色标签。它并非简单地悬浮在卡片上空，而是与白色卡片**发生物理粘连，呈现完美的不倒翁外扩弧线曲面 (Swooping Base)**。
为了在严格的“零 SVG / 零图片”约束下还原该异形边界：
- 采用 `.card-top-tab` 设置 `border-radius: 14px 14px 0 0` 定位于绝对位置 `bottom: 100%`（即骑载在白卡的顶部边界沿线）。
- 左右两侧释放了两个极度隐蔽的伪元素 `::before` & `::after`。它们被固定在标签根部外的透明正方形内，并设置了 `border-bottom-radius` 的切角。
- 最巧妙的是应用了 `box-shadow: 4px 4px 0 4px #fff` 的投射！利用这股强力的白色阴影向上填满透明框，从而把母体白卡的躯干顺理成章地“拔”向橙色标签的夹角，构成了完全无缝连接的贝塞尔倒角体验。

### 2. 斑马纹记账本底纹 (Number Ledger Grid)
在巨大的红色 `1.6530%` 数字背后，漂浮着密集的水平账单划线。
- 采用 `.number-ledger-bg::before` 生成绝对定位遮罩。
- 引入了精密计算的周期线性渐变背景引擎：`linear-gradient(rgba(240,240,240,0.8) 1px, transparent 1px)` 和 `background-size: 100% 12px`，在 0 额外渲染开销的情况下，源源不断地生成了具有金融安全暗示特征的防伪细线栅格。

### 3. Tailwind 原生图标阵列 (SVG Path Vectors)
顶部的 11 宫格没有引入任何外部 `<img>` 依赖。完全依靠将通用的金融服务隐喻转换成精炼的 SVG 纯几何 Path：
如 `ChartSquare`（走势）、`LeafCoin`（应急储备）、`ShieldLock`（账户安全守护）、`Ticket` 等。通过向这些矢量填充与父级文本同出一脉的色调，保持了原图中极为规整的数据陈列仪态感。
