# 投顾发车 (page-08) 规格文档

## 视图说明
本页面为京东金融 APP 内部相对专业的子功能页：“投顾发车”。为用户提供结构化的定投、跟投操作指导及资产分配建议情况。因此底部的导航栏完全异化，切换为了专用于该业务的 5-Tab 栏。

## 布局结构解析

1. **背景区域**
   - 包含了一个自上向下的混合过渡色系由柔和的浅肉粉降至标准的灰色背景底框 (`#f6f7f9`)。
   - 顶部右上方带有倾斜绝对定位的装饰性大箭头纹理底纹（应用了通过 CSS Clip-path 和极低不透明度模拟的水印 `watermark-arrow`）。

2. **顶栏及标语区 (Header & Banner)**
   - Header 呈现标准的左返回箭、右主菜单 `...` 功能布置。
   - **标语**: “投顾发车一路相伴”以偏棕红的极粗字号展示。下挂三个带打勾的辅助卖点（“紧跟市场”，“悉心择时”等）。右侧浮置一个毛玻璃透出背景的导向药丸按钮。

3. **核心发车状态指示牌 (Status Card)**
   - 第一张卡带红色火苗图标标题：“正在发车”。
   - 提供由 3 列构成的快照数据展现：第一列对应“上证指数”及动态红涨率；后两列带有极浅淡的红色填充模块，分别说明组合数量及当前的预警/投入信号（如：19个，正常投入）。

4. **跟车标的列表块 (Fund List Cards)**
   - 定义了一个子 React 组件 `<FundCard>`，统一渲染后续所有投资组合（“牛气冲冲”、“福气满满”、“司南全球多元精选”）。
   - **交互点**: 右上方为鲜红的突显诱导按钮：“去上车”。
   - **结构层**: 包含一整行 3 列的动态关键数据，例如发车日、操作倍率（通常会加大字号加粗渲染）以及带有红色高亮提醒的倒计时（距结束）。
   - **操作意见区**: 底部新增了由 5 层块状进度组成的“主理人观点”（横向渐进指示器）。当前渲染使用了 CSS 分配机制，满分为 5 满红色（对应乐观）。“司南”对应着 3 格。下方分别标注：谨慎、偏谨慎、中性、偏乐观、乐观。

5. **专属功能悬浮底导 (Dedicated Bottom Navigation)**
   - 底部全局浮层的结构并未延用金融客户端首页的常规 Tab。
   - 分别更迭为：首页 (Home)、投顾 (Advisor Silhouette)、发车 (Car outline)、自选 (+ icon)、持仓 (Coin symbol)。
   - **发车**：此项处于最高重选状态（字体居中加黑加粗，图标切换为正向渲染的汽车头造型的黑色 SVG 图形）。
