# 白条权益页 (page-70) 规格文档

## 视图说明
`page-70` 为京东金融“白条权益”会员特权落地页。
页面视觉传达出非常强烈的钻石冻龄/冰晶质感（Ice/Glass Theme），展示用户的 V1-V6 级别体系以及匹配的权益卡片、票据化的特权说明。由于全站要求零资源开发，本页面的诸多 3D 切图均使用几何重组与级联滤镜替代生成出强拟真效果。

## 零资源前端重构技术概览

### 1. The Ice Stage (冰川环境场 & 进度光轨)
上半部分屏使用渐变脱焦环境营造空间景深：
- 页面底图采用了倒置扩展的 `linear-gradient` (`#dbefff` -> `#f4f6fa` -> `#f4f5f7`)。
- 在 `V 进度轴` (`.v-line`) 的中心线，我们并没有简单的一条实线跨过，而是利用 `linear-gradient(to right, transparent, #8ab5e3, transparent)` 进行首尾 Fade-out，这产生了一种高精度的 HUD 激光扫描质感。同时每一个 V 字节点背后也覆盖有相同背景色的方形色块，通过物理覆盖遮挡从底部穿过的连接线，避免连接线穿插文本带来脏感。

### 2. 伪 3D 几何切割 (Hexagon Frame Mockup)
在原始设计中，V2 尊贵卡片右侧卧躺着一只戴皇冠的 3D IP 大鹅。为了在不引入任何大体积 png / webp 图像的前提下保证此区域不会缺失导致视觉塌陷：
- 应用纯 CSS API `clip-path: polygon(50% 0%, 100% 25%, 100% 75%, 50% 100%, 0% 75%, 0% 25%)` 精准裁切出双层正六边形，通过 `#aed6f8` 等冰川蓝颜色的不同坡度渐变，复刻出冰雕底座。
- 在内部辅以系统级基础 Emoji 组合 (`👑` + `🦆` + `💎`) 及巧妙的纵深向相对定位偏移 (`top`, `z-index`, `transform`)，构成一幅低俗多边形风格（Low-Poly-ish）的高级嘲讽趣味头像站位替代符。

### 3. Pinhole Tickets (打孔券板体系)
“月月领”内的优惠卡券视觉极其清晰。对于卡券两侧的物理“撕口”或“打孔”特征：
通过 `.coupon-ticket::before` 和 `::after`，给两边分别放置 `6px` 的小圆，并设置其背景色完全等于它所属物理区域后方的基底色 (`#ffeeea`)，同时仅开启半侧的 `border-right: 1.5px solid #ff4e4e;` 使其线条与外框相连，完美“骗过”视觉系统，这使得我们的票券能够无限动态缩放而永不失真。
