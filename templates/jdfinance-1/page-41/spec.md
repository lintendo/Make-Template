# 抽奖弹窗结果动画页 (page-42) 规格文档

## 视图说明
`page-42` 并非一个绝对独立的物理页面，它是架设在 `page-41`（签到领现金大厅）正上方的全屏动画弹框（Lottery Results Modal）。
由于我们在此开发阶段采用强制物理文件隔离原则（Standalone Node），此组件需要同时展现出后方被调暗（`backdrop-blur`）的底图轮廓，以及处于 Z 轴顶层的中奖狂欢动画特效。

## 零资源前端重构技术概览

### 1. 背景底层幻影复刻 (Ghosted Underlayer)
原有的设计稿背景是被一层处于 70% 透明度下的纯黑遮罩蒙住的签到页体系。
为了满足孤立原型测试的要求而不污染主组件，我在背景搭建了一套由 `opacity-30 blur-[1px]` 生成的抽象签到大厅轮廓架构。它以纯粹的色块（Fake Cards）结构存在于底层，加上 `.bg-black/60` 的 Overlay 形成极强的高对比夜间深邃视感，突出了中间发亮的宝箱实体。

### 2. 多重渲染宝箱本体 (Emissive Reward Box)
页面最核心的信息是“0.01元”现金打款宝箱卡。这不仅仅是一个放大的卡片：
- **光晕发光体 (Green Emissive Aura)**: 卡片并不是贴着黑背景存在的，在其正后方我运用了 `radial-gradient` 且外挂了无限循环的 `animation: auraPulse 2s` 指令，渲染得到了一个如同呼吸般吞吐扩张的刺眼绿色光晕系统（Aura）。
- **复合阴影装甲 (Composite Box-Shadow)**: 针对容器模型 `.mega-modal-box` 本身引入多重阴影，包含了向下的黑色大投射环境光（`0 20px 45px rgba(0,0,0,0.4)`），向卡片切角的白色倒边高光（`inset 0 3px`），从而构建极其高级感的三维渲染材质。

### 3. 庆祝粒子系统 (CSS Confetti Engine)
在没有任何外部 PNG/GIF 的环境下需要满足欢庆特效的诉求。
- **粒子发生器**: 封装了 `floatUp` 关键帧动画引擎。并在全屏散布带有不同延迟（duration & delay）的 CSS Emojis （❤️, 🪙, ✨）。它们会独立依照弹道随机散落并且向上抛射并逐渐变淡，再现了烟花散射的状态。
- **抽象鼓掌几何体**: 屏幕最下沿的 3D 卡通手势属于不可复刻图像模型。在此进行了解构与重铸化，部署了4条倾斜组合排列的红色-肉色渐变流线型立柱体系 (`.abstract-hand`) 组成了一个极为意识流且动感十足的抽象手部庆贺群。

### 4. 黄金3D文本映射 (Extruded Golden Typography)
- 顶部浮动的巨型 "现金打款" 使用了 `.mega-gold-text` 机制。这绝不是一个平面文本。不仅表面通过 `linear-gradient` 进行 Webkit 文字裁剪赋色。还在它的背部使用 `::after` 附着了一层具有同样文字但偏褐色并不具有文字发光的克隆底模，并下移 `2px`。通过这种双文本栈重合渲染技术硬切出了3D金属字体的物理挤压截面体积感效果（Thickness Extrusion）。
