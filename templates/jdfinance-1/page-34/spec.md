# 基金持仓详情页 (page-35) 规格文档

## 视图说明
本页接管了 `035-pic` 的“持仓”（Holdings）分页状态。
在此页面中，系统要求将庞杂的资产分布情况进行高保真的数据可视化呈现格式，并且展示出一个包含微缩指示箭头的基金列表。我在此页面深度挖掘了 CSS 的复杂排版组合能力。

## 数据可视化组件开发

1. **三段式全量环形图 (Tri-color Full Conic Donut Chart)**
   - 核心任务展示 88.44%, 7.45%, 4.12% 的三段切分扇形图。
   - 实现方案为 `.full-donut-container` 内部运用硬极角过渡：
     `background: conic-gradient(#ff3a48 0deg 318deg, #ffa24b 318deg 345deg, #ffcd50 345deg 360deg)`
   - 以上逻辑将圆弧按比例完美切分成红色、橙色、黄色三部分。
   - 随后置入 `.full-donut-cutout` 原理一致的核心遮挡层来挖出空心圆，内部承载“净资产”的静态数值展现。没有任何外部库负荷。
   - 为图表赋予 `box-shadow: inset 0 2px 4px rgba(0,0,0,0.02)` 营造平滑内陷的圆润感。

2. **自适应伪表格体系 (Pseudo-Table Financial Grid)**
   - 在移动端开发标准 `<table>` 是极为沉重的。我启用了 `flex-[X]` 对比弹性盒布局引擎制作了完全对齐的列矩阵框架。
   - 在表头部分：`flex-[0.38]`, `flex-[0.24]`, `flex-[0.16]`, `flex-[0.22]` 完美平衡了基金名称（定宽换行）、涨跌幅（居中对齐）、占比、较上期（靠右带图标）的空间诉求。
   - 行单元使用了浅灰 `.bg-[#f8f9fb]` 及点击下沉 `active:scale-[0.98]`，实现了高级原生应用的操作阻尼感。

3. **CSS 矢量金融符号 (CSS Vector Up-Tick Triangle)**
   - 对于“-0.07%”采用了中国式的跌幅色彩（绿色）。相应的，对于右上角小股评的较上期 “+3.54%”，我采用纯粹的底边长边境宽等腰三角形 `.triangle-up-red`。
   - `border-bottom: 5px solid #fc4a4f`, 配合 `border-left: 3px solid transparent`, 做出了不带任何 SVG 包涵量微型向上的红色箭头。

底栏导航区域继承了上文所述的极端复杂矢量组件切割（PK / Mascot 等重构图标）。保持了组件体系的一致性与高响应度。
