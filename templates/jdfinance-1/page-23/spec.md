# 农户借款 (page-24) 规格文档

## 视图说明
本页面为京东金融APP旗下针对农业用户的特色借贷专版：“农户借款”面板控制台页面。
该页面的核心业务是为高度授信的用户提供最高达 20万量级的极速免抵押放款服务呈现，并通过特殊的视觉表现传达其作为三农金融产品的定位。

## 布局架构与核心功能复现

1. **三农业态沉浸式背景 (Agrarian Hero Background)**
   - 彻底废除了传统的科技蓝/金融红，运用了具备极强联想属性的深麦绿色 (`bg-[#12452a]`) 并在 `style.css` 中注入 `.farmer-hero-bg`。
   - 这套定制版 CSS 使用了两个高度抽象并且带有通透度 (`rgba`) 的 `radial-gradient` 黄金光源组合，打在第三层深绿色线性渐变上，不用加载任何一张写实农田照片，依靠存粹的色光组合就能 100% 模拟出“黄昏稻田”光照质感。

2. **核心借贷卡片 (Main Financial Hub)**
   - 设计为拥有极高投影 (`shadow-[0_4px_25px_rgba(0,0,0,0.06)]`) 和超大圆角 `rounded-2xl` 的白色核心悬梯。
   - 在卡片的背景之中，我们插入了极其隐蔽的 `absoulte` 定位空环线 (`border-dotted`)，以此在 DOM 级别去重构背景里淡淡的麦穗/稻谷抽象纹理而无需求助外部图片加载。
   - 借款金额字体 `font-normal text-[64px] tracking-tight` 加上下方大面积的警戒色标签块 `bg-[#fef4ea]`，最大化信息阅读层级。

3. **定制化营销卡牌 (.promo-ticket-border)**
   - 包含着“降息红包卡”等营销玩法的容器不仅使用了柔性的暖橙色打底，最关键的工艺体现在其边框工艺。
   - 我们在 `style.css` 部署了非常苛刻的多重伪元素 (`::before`, `::after`) 与绝对定位类 (`promo-corner-tr`)，利用部分隐藏边框 (`border-left: none` 等) 把正方形切分为“直角 L 型角缀”，精准地挂在边框的四个死角上，在 CSS 层实现了 0 延时渲染的复古优惠券装裱感。

4. **硬核底部 TabBar**
   - 新增页面级通用低代码 `TabBar`，吸底悬浮并且使用 `shadow-[0_-2px_15px_rgba(0,0,0,0.02)]` 防遮挡。包含手写的微型 SVG 极简 Icon 充当“还款”、“我的”功能导流口。
