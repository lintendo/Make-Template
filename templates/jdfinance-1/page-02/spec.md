# 理财资产页 (page-02) 规格文档

## 视图说明
本页面为京东金融的“理财资产”页面，呈现用户的资产持有情况及相关理财推荐服务。整体背景由顶部的浅粉色渐变过渡到温和的浅灰背景。

## 布局结构

1. **顶部导航栏 (Top Navigation Bar)**
   - **返回按钮**: 左侧深色返回箭头 (`<`)。
   - **标题**: 中央字体加粗显示“理财资产”。
   - **客服/耳机图标**: 右侧显示，图标带有红色未读/提示小红点。

2. **核心区块 1: 持仓概览 (Position Overview)**
   - **头部标签**: 标题“持仓概览”右侧附带橙色描边的“资金保障中”小标签。右上角有渐变表现形式的“财富会员 Lv.1”皇冠标签。
   - **资产数额**: “我的资产(元)”配合隐藏金额的“小眼睛”图标。主数字 `0.00` 使用极大字号呈现。
   - **明细划分**: 底部分为左右两列，分别为“昨日收益”与“累计收益”，数额下方由向下折叠的灰色小箭头收尾。

3. **横幅入口: 稳健理财 (Banner)**
   - 插入式的白底通栏卡片，左侧配有极细红色背景药丸标签“稳健理财”。
   - 核心文案：“学知识领福利，追求稳稳正收益”。附加右向小箭头引导点击。

4. **核心区块 2: 资产明细 (Asset Details)**
   - 两栏两行的列表网格结构。
   - 包含“京东小金库”、“基金”、“京东活钱+”等具体资产配置的明细或推广文案，例如“小金库待提现”、“买基金费率低至0%”等。

5. **核心区块 3: 专属服务 (Exclusive Services)**
   - 提供两个对称的增值服务卡片：
     - **风险类型卡片**: 浅蓝紫色渐变背景，中间放置蓝色 3D 质感 A1 盾牌图标，指示用户为“保守型”，并建议“适合低风险产品”。
     - **京东全天候卡片**: 浅粉红渐变背景，以饼图展示资产配置（如60%全球投资，20%全球债券，20%大宗商品），使用不同红色系区块及连接指示线进行了还原表示。

## 交互设计
本稿作为 UI 层高保真还原，目前组件包含折叠箭头、资产显隐开关（眼睛图标）和条目入口箭头，均为功能性触发入口样式的静态表现，供后续交互扩展使用。
