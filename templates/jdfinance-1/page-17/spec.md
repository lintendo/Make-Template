# 空白订单页 (page-17) 规格文档

## 视图说明
本页面为京东金融APP通用功能页面的标准空状态 (Empty State) 缺省图。
主要场景为用户进入个人订单历史且数据量为零时触发。它承载了平复用户可能因为“查无数据”产生的负面情绪的作用，故使用 IP 形象卖萌占位。

## 布局结构与特殊重构说明

1. **基础导航容器 (Header)**
   - 具备居中双核心 Tab 切换能力（“话费订单 / 会员订单”）。
   - 激活态 (“会员订单”) 采用大字重黑色匹配短而圆润的京东红底部下划指示条 (`bg-[#ea3d33]`)，非激活态保持静默灰色。保证视觉中心集中于激活态。
   - 使用了原生 `sticky top-0 z-50` 约束结构，以兼容可能产生滑动距的场景。

2. **核心业务：防腐层“微插画”映射策略**
   - **技术重构亮点**: 为了将 1:1 复刻的追求贯彻到底，并且保持绝对的网络请求解耦和体积压制，我们没有去提取/切取 `017-pic` 中的京东小白狗位图！
   - 相反，我们利用了 **完全原生的高级 CSS 层叠与 SVG 联合渲染技术** 动态手写出了这只虚拟宠物！
   - **分层策略 (JD Dog DOM Hierarchy)**:
     - *环境层*: 粉色聊天气泡背景底座。通过利用线性渐变辅以特定的百分比变形 (`borderBottomRightRadius: '10%'`)，使其自然塌陷形成对话框尾巴。
     - *主体头部 (`Dog Head`)*: 白色圆润矩形打底，配以轻微的扩散投影，增加厚重度感。
     - *耳朵组件*: 将纯白椭圆形通过 `transform: rotate` 以及轴心移动 `origin-top-*`，完美还原垂眉搭眼的丧感和可爱感。
     - *五官 (`Facial`)*: 调用了最轻量级的内联 `<svg>` 矢量直线勾勒出了 `> <` 紧闭的眼睛轮廓；利用堆叠组合 `border-radius: 50%` 构建出了张大的红色口腔区域和下沿阴影。
     - *躯干与双臂 (`Body & Arms`)*: 设置双层 Z-Index 交错的白胶囊形模型构建高举的双手，配合底部模拟环境遮挡阴影。
   - **意义**: 使得整个空状态图表不仅具备了绝对的高视网膜屏匹配度（无限放大不失真），同时由于全静态 DOM 构建，完全规避了传统应用在渲染空屏提示时还要进行多频次 HTTP 网络图库请求的心智负担，做到秒开。

3. **空态提示语 (Typography)**
   - 正下方跟随 “还没有充值记录” 文本，排版字距被微调放宽，颜色提取为 `gray-500` 低对比度灰色，传达出柔和的陈述属性，而不过分干扰用户视觉。
