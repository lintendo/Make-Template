# 缺口分析 (page-28) 规格文档

## 视图说明
本页面是“家庭应急金”板块的第二个全局 Tab 导航落地页：“缺口分析” (Gap Analysis)。
该页面放弃了传统的列表导购形态，转而使用充满压迫感与急迫性的沉浸式红橙巨型背景，通过个人资产缺口雷达式的数据测算，强烈驱动用户一键启动体验资格。

## 布局架构与微元素复原说明

1. **贯穿式深空渐变画布 (Full-Bleed Hero Gradient)**
   - 使用 CSS `.gap-hero-bg` 构建从头部贯穿至页面大屏的 4 阶渐变 (`#ff5715` 到 `#fef3ef`)。
   - 这摒弃了过去“只有顶部背景+下面全白背景”的通用套路，带来如同独立小程序般沉浸的前置营销感受。

2. **高保真数据仪表板 (Data Dashboard Replicas)**
   - 顶部的“缺口测算”卡片使用 `.gap-card-gradient` (`#fffbf6` to `#ffecd8`) 取代单调的白色。
   - 内部包含了垂直错位的浮动式选择器（`北京市 ▼`）、带有虚线的分割线框 (`border-dashed border-red-300`) 以及 34px 巨大的 `万元` 不对等错位文字设计，这些均为金融数据面板的压轴细节。

3. **家庭成员流式交互签 (Fluid Family Tag Interactions)**
   - **状态指示器下挂**: `.family-active-tag` 巧妙应用了 `::after` 极简绘制规则，在其正下方打出一个居中对齐的红色实心向下箭头，直指下方的测算结果。
   - 动态横铺了“本人”，“父亲”，“母亲”及“添加”等标签，各自带有不同的边框与背阴影样式。

4. **矢量插画渲染引擎 (CSS Avatar Generation)**
   - 为避免加载位图和解决高清屏幕适配问题，我在“治病缺口”展示框内手动利用 `radial-gradient` (.dad-avatar-bg) 加上 3 层嵌套 `div` (呈现出地中海发型与复古粗框眼镜)，在完全脱离图片素材库的前提下构建出了高精度的父辈像素风人像，将组件内部内聚度拉满。

5. **嵌套多边形连带菜单 (Chevron Breadcrumb Tabs)**
   - 底部嵌套的产品配置 Tab (`0元体验`, `惠民医疗`...) 在设计上首尾相连。
   - 为实现这种特殊的首尾咬合感，在 `.chevron-tab` 中激活了高阶 `clip-path: polygon` 指令切出精准的横向尖角，并通过 `margin-right: -6px` 使每个标签相互物理重叠吞噬空白边界，重制效果 1:1 惊艳。
