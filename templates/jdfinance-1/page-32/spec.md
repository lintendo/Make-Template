# 基金档案详情页 (page-33) 规格文档

## 视图说明
本视图对应着原图片 `033-pic`，这是典型的极简垂直金融属性文档页：“档案”。
在此页面内，屏弃了重型的花哨动效营销光影，所有的重心都落在严肃的信息架构（Information Architecture）对齐。主要构建挑战在于跨度极大的时间轴图形构造、微型边角徽章处理以及巨量功能图标的扁平化重构。

## 组件工程解密

1. **图文级节点时间轴 (Timeline Component Map)**
   - 在“交易规则”下的买卖路径地图是由底层水平绝对跨越线条 (`absolute .top-[6px] .h-[2px] bg-[#e3e6ec]`) 串联三大核心节拍点 `.timeline-node` 构建的。
   - `.timeline-node` 是极其纯粹的几何：利用原生的 `border: 3px solid` 环搭配 `::after` 虚伪类的同心圆红点注视，从而取代了加载第三方步骤条组件。
   - 为了打破僵化的 33% 间距分布，第一阶段与末尾阶段分别启用了 `w-[80px]/items-start` 与 `w-[130px]/items-end`，中间使用 `flex-1` 膨胀顶挤，完美复原了原稿不对称却和谐的对齐感。

2. **浮动侧切标 (Edge-Snapped Floating Tab)**
   - 悬于右侧的“调研问卷 X”：直接在 React 端使用了 `fixed right-0`。因为文本方向垂直，我们利用了核心 CSS 书体映射 `writing-mode: vertical-rl` 及 `text-orientation: upright` 。这是一个极为老式但强悍的排版属性，成功避免了让字符翻转 `-90deg` 致使用户阅读扭脑袋的错误反直觉体验。

3. **复杂金融图表重写 (Vector Financial Icons Rebuild)**
   - 底栏 4 枚灰色工具按钮看似简单大方，其实暗藏魔鬼像素细节。
     - **理财助手** (Robot)：复用了早前的小京灵 Mascot 构建逻辑，压缩到了惊人的 `22px` 级别。
     - **加自选** (+)：直接在空框 Div 内注入原生 `+` 字幅并进行 baseline 核准。
     - **对比PK**：这是一个切除了底部一边直角的“半盾”形状。在这里直接利用 `border-b-lg rounded-t-[3px]` 切出了外轮廓，内部通过文本 `-skew-[15deg]` 造就倾斜感硬派竞技视觉。

4. **微距特殊字体处理 (Font Scalar FA Logo)**
   - 头部导航栏处的缩放字体徽章 `FA` 并非图片。采用了父容器 `border-[1.5px]` 裹挟着异尺寸文字，并利用原生的 CSS 边框切割 `w-0 h-0 border-l-[6px]` 制成徽章折角的极酷纯前端解围思路。 

这种极端克制无外联图片的实现策略保证了哪怕在 4K 屏幕上，所有的边缘将如初生婴儿般平滑。
