# 京东信誉分 (page-22) 规格文档

## 视图说明
本页面为京东金融APP针对账户风险评估与增值服务建立的核心门户：“京东信誉分”面板页面。
主要承担了积分即查即显，权益转化，以及推销“增加分数值的任务系统”的大型漏斗口功能。

## 布局体系与核心架构重建

1. **绝对量化数据展示盘 (The Glowing Hero Dial)**
   - 此模块视觉要求极高，传统框架常依赖一整张 `.png` 质感贴图叠底，这会导致原型放缩模糊。
   - 重建技术方案：
     - **底层背景 (`score-hero-bg`)**: 使用 `radial-gradient` 极坐标复合映射出 3 个维度的光源交叠态，复现京东红的光晕过渡效果。
     - **环绕记分仪表盘 (`SVG Data Arch Engine`)**: 抛弃传统 `border-radius: 50%` 再切掉一半的笨招，直接调用 **原生 HTML `<svg>` 构建了一个精确到像素点的多层同心环** (`M x y A r r ...`)。
       - *内层点状轨迹*: 定义了 `<path>` 通过属性 `strokeDasharray="3 5"` 与透光 `rgba` 直接计算出绝对均匀的环形虚线！
       - *外层导电轨距*: 使用 `<linearGradient>` 给高光线描加塞了首尾弱化的渲染能力，模拟真正的光源抛光反射弧。

2. **特权实体券分离器 (`.privilege-ticket`)**
   - 之前页面（充话费）运用过的边缘打孔 CSS 黑科技，在“月度权益”横向滚动区域得到升级。
   - `privilege-ticket-divider`: 除了左右分离使用 `dashed` 断裂带以外，该 CSS 规则利用 `::before`/`::after` 把下钻孔（背景色的透明咬合口）锚定在了两张券的核心交接面，提供了百分百真实纸质券撕裂纹的交互物理基础。

3. **微型卡片阵列 (Miniature Vectors)**
   - `精选` 服务区的所有图标和任务管理列表中的“高级包/职业证明”图标等，一如既往采取 DOM 层全手刻化。
   - 例如第一张卡片的绿色小书包【先享后付】图标：使用 `border-[2.5px] border-b-0 rounded-t-full` 生成完美半把手后，与底层的绿软块拼合构成抽象物理形态；【灵活还款】更是灵活运用绝对下潜 `z-0` 跟主卡片主体叠穿打造抽拉质感。

这确保了即使是复杂图表结构，工程项目依然零依赖极速渲染。
