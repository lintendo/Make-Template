# 金条借款主干操作页 (page-68) 规格文档

## 视图说明
`page-68` 展示了京东金融“金条借款”全流程的底部主级页面 (Main Control Dashboard)。也就是前三个截图 (page-65, 66, 67) 退除底层遮罩层后的核心系统。
整个页面分为大金融核心区（贷款展示栏）、主交易卡版（借款条款调控面板）、服务合规卡版（次级设置）以及底部悬浮固定的提款核准动作组。

## 零资源前端重构技术概览

### 1. 拟物化金融表单卡片体系 (`.jd-card`)
全面摒弃硬边界线，使用现代金融 APP 主流的区块分割方案：淡灰底 (`#f7f8fa`) 上铺设卡片容器。
- 使用 `margin: 0 14px 12px` 与 `border-radius: 12px` 来产生具有厚度但不沉重的白色卡版组。
- 内部列表 `.list-row` 组件采用了纯粹的 CSS Flexbox `justify-between` 两极拉伸布局，使得右侧所有向右的箭头标致 (`chevron-right`) 做到了完美的纵向对齐，不需要刻意设宽度便可根据内容撑开。

### 2. Tab Navigation & Strike-through Topology (内嵌页签与财务注销线)
首张主卡内部有着重构难度不低的复杂标签页切换头。
- **布局切分**: 放弃简单的按钮做法，使用 `.tab-column` （纯 Flex 均分布局）并在两个列阵中间放置了一条物理高度确切的竖线分隔符 `.tab-divider`。
- **内嵌微型排版**:
  针对“单利 16.1% 以及失效划线价 18.3%”：我组合使用了行内 `font-sans` 数字字体保护、以及 `.line-through opacity-80` 对失效过往年排版的渲染叠加，不需要输出图片素材便完成了精细营销价格阶梯的对比感构造。

### 3. Glassmorphic Sticky Button Base (毛玻璃固定交互底槽)
为防止巨大的“确认借款”行为被内容滚动遮挡。使用标准的底部吸附：
- `position: fixed; bottom: 0; w-full` 将整个操作台拉满视口防越界。
- 应用了苹果级渲染特性 `background-color: rgba(255, 255, 255, 0.98); backdrop-filter: blur(8px);`。它不仅仅是一块白布，随着用户滚动底部卡组，页面的文字与卡片阴影在穿透该悬浮面板时会产生极微的朦胧融合，将原型保真度上推至 Native 客户端级别。
