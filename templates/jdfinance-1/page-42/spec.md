# 签到领现金强提醒弹窗页 (page-43) 规格文档

## 视图说明
`page-43` 呈现的是应用端“每日签到打卡”系统的高优先级强阻断式弹窗（Force Alert Modal）。
该页面的使命是通过巨大的动效和强烈比对的色彩迫使用户完成每日点击任务，防止连签断档。在技术实现上，我同样沿用了“底层背景轮廓克隆”+ “重度黑色遮罩蒙版”的主架构策略，使得我们可以在脱离主工程的独立路由下测试和验收弹框 UI。

## 零资源前端重构技术概览

### 1. 翡翠流星球体重构 (Emerald Orb Node)
原本该界面的上方会漂浮一个具有极亮绿光的领奖球。这也是该 UI 的视觉锚点（Anchor Point）。为了在零依赖前提下逼真呈现此发光效果，进行了以下几何实现：
- 启用了基于 `.emerald-orb-glow` 类的复合渲染阵列。在基础 `radial-gradient` 之上加入外扩散发光层（`box-shadow: 0 0 45px rgba`）与反向内发光层（`inset -5px -5px 15px rgba`）。
- 给圆形添加最外圈实边高光（`border: 4px solid #fefce8`）。并运用绝对定位 `top-[-55px] left-1/2` 机制，将其悬挂骑跨在主骨架白色大模态框的顶格中轴线上，完成物理阻挡与跳离视网膜效果。

### 2. 动感排版与金币几何映射 (Kinetic Typography & Geometry)
- **非线性标题 (`.slant-title-text`)**: 标题不仅加粗放大，还通过设置 `transform: skewX(-12deg)` 赋予强烈的倾斜物理动量（Forward Momentum），模拟了电竞或大促海报的拉拽冲击力。
- **纯 CSS 3D 金币 (`.css-gold-coin`)**: 运用细致叠加的多重径向渐变色（Radial Gradients）并辅以黑色边框映射与内置文字偏移，手动雕刻出了位于水晶球左下方与右下方悬浮的小直径人民币纯金币装饰图素。它们被分配了相反的倾斜角，从而在无动画静态也实现了活泼的重力感。

### 3. 主券面质感与角标悬浮 (Prize Ticket & Badges)
中央核心区的“0.52元橙色代金券”使用 `.prize-ticket` 进行色彩切割。上半身纯白展示数据，下半部承接火焰式渐变。在右上方安插了不受 DOM 文档流约束的悬浮角标“明天可提现”，突破父级轮廓，有效刺激眼球视线。
