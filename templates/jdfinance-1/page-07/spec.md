# 省钱频道附加奖励模态窗 (page-07) 规格文档

## 视图说明
本页面为京东金融的“省钱”版块的附加营销型大弹窗视图，主要在用户进入频道或满足触发阈值时弹出。主要目的是利用积分概念（京豆）刺激用户行为并进行跨频道引流。

## 布局结构

1. **底层背景映射层 (Mock Background App Frame)**
   - 使用 CSS `opacity-30` 及指针禁用 `pointer-events-none` 模拟出被压在弹窗底层的原始页面内容（包含了搜索栏结构与底部省钱 Tab 亮起的全局导航栏）。

2. **核心覆盖层 (Modal Container)**
   - 全屏绝对定位层级置顶，顶部预留了约 `16vh` 的视觉缓冲空间。

3. **右上侧辅控按钮**
   - 带有毛玻璃效果（`backdrop-blur`）及白色透明描边的环状药丸按钮“跳过”（Skip Button）。

4. **主视觉文案区**
   - **标题**: “省钱频道 奖励升级”。
     - 前方字体应用了特殊的纯白填充与灰色粗描边属性。
     - 后半部分文字运用了发光金色的 `text-shadow` 加叠文字流光，打造奢华提示感。
   - **氛围挂件**: 使用绝对定位排布各种彩带与氛围光点（五颜六色的 CSS 方块小形变），并加上了背景大号的高斯模糊黄光托底以凸显整个金钱主题。

5. **京豆收集横向卡片区 (Rewards Ribbon)**
   - “一键领取”（红色按钮）位于右侧。
   - 四个排列在一起的任务节点槽：展示了一颗基于纯 CSS 绘制的不规则带阴影高亮发光豆子（`jd-bean-small`），每一个都带有不同红黄标识的补贴数字角标（如 `+2`, `+5`, `去虎牙 +1` 等）。

6. **巨型说明图文组合 (Diagram Graphic)**
   - **积分点框 (左)**: 一个浅黄渐变框（底部带硬阴影）。内有金币图案，配字“积分”。
   - **跨界红箭 (中)**: 自定义的 SVG 曲线连贯箭头，填充为红白两色的双重渐变，用来表达积分转换或资产重叠的物理动态感受。
   - **京豆兑换主导框 (右)**: 体积更大的偏白盒身带金框顶，顶端放置两颗巨大化且叠加了内光圈、外投影的 CSS 绘制“京豆”。大字主体标明“京豆”，带有比例贴纸及长尾补充提示文字。

7. **导向结束区 (CTA Button)**
   - 底层配上了发光的巨大行动圆角按键：“开始体验”横绝屏幕下方核心触点。
